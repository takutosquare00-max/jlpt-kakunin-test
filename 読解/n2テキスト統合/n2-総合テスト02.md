## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　チケットを買う際は、身分証明書の <u>提示</u> が必要だ。
1. ていし　　2. ていじ　　3. だいじ　　4. てんじ
> **正解：2（ていじ）**　「提示」＝見せて示すこと。「提」＝テイ、「示」＝ジ。「ていし」「てんじ（展示）」と混同しないよう注意。

**2**　日本で映画を見るには、<u>通常</u> 1,800円ぐらいかかる。
1. とうじょう　　2. つうしょう　　3. つうじょう　　4. つうじょ
> **正解：3（つうじょう）**　「通常」＝ふだん・いつもの状態。「通」＝ツウ、「常」＝ジョウ。「とうじょう（登場）」「つうしょう（通称）」と読み間違えやすい。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　この映画館では、館内で毛布やクッションを（　　）するサービスもある。
1. 貸し出し　　2. 売り出し　　3. 呼び出し　　4. 乗り出し
> **正解：1（貸し出し）**　「貸し出し」＝物を一定期間貸すこと。図書館や映画館などのサービスで使う。「売り出し」は販売、「呼び出し」は人を呼ぶこと。

**4**　夫婦のどちらかが50歳以上の場合、料金の（　　）が受けられる。
1. 値上げ　　2. 割引　　3. 請求　　4. 支払い
> **正解：2（割引）**　「割引」＝決まった値段から安くすること。「値上げ」は逆に高くすること、「請求・支払い」はお金を求める／払う行為で文脈に合わない。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　この体操は体力（　　）、無理のないように行ってください。
1. に限って　　2. に応じて　　3. にかわり　　4. において
> **正解：2（に応じて）**　「に応じて」＝状況や程度に合わせて。「体力に応じて」は自分の体力に合った形でという意味。「に限って」は〜だけ、「において」は場所・場面を表す。

**6**　水泳は子どもからお年寄りまで世代（　　）楽しめるスポーツだと言われている。
1. ばかりでなく　　2. に限って　　3. を問わず　　4. とおりに
> **正解：3（を問わず）**　「を問わず」＝〜に関係なく、どんな〜でも。「世代を問わず」はどの世代でも関係なくという意味。「に限って」は〜だけ、「ばかりでなく」は〜だけでなく。

**7**　ただ今、こちらの商品（　　）、全国どこでも210円で配送を承ります。
1. を問わず　　2. の際は　　3. に限り　　4. に応じて
> **正解：3（に限り）**　「に限り」＝〜だけ（限定）。「この商品に限り」はこの商品だけという意味。「に応じて」は程度に合わせる、「の際は」はそのときは、という意味。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　＿＿ ＿＿ <span class="star">★</span> ＿＿ 確認しておきましょう。
1. 際に　　2. 地震の　　3. 避難できる　　4. 場所を
> **正解：3（避難できる）**　正しい語順：「地震の→際に→★避難できる→場所を」。完成文：「地震の際に避難できる場所を確認しておきましょう。」「〜の際に」＝〜のとき（改まった表現）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　日本で映画を見るのに、通常1,800円ぐらいかかりますが、映画館では、通常料金のほかに、特別なサービスがあります。
>
> 　毎月１日の「映画の日」は、年齢・性別を問わず誰でも1,000円です。水曜日は女性に限り1,000円になる映画館もあります。また、夫婦のどちらかが50歳以上の場合は、いつでも2人で2,000円です。ただし、チケットを買う際は、身分証明書の提示が必要です。
>
> 　そのほか、館内では、毛布やクッションを貸し出すサービスもあります。

**9**　この文章の内容に合うものはどれか。
1. 映画はいつ見ても1,800円である。
2. 「映画の日」は、年齢や性別に関係なく誰でも1,000円で見られる。
3. 水曜日は男女ともに1,000円になる。
4. 夫婦割引を使うには、必ず2人とも50歳以上でなければならない。
> **正解：2**　本文に「毎月１日の『映画の日』は、年齢・性別を問わず誰でも1,000円」とある。水曜日の割引は「女性に限り」、夫婦割引は「どちらかが50歳以上」なので3・4は誤り。

**10**　チケットを買うときに身分証明書が必要なのは、どんな場合か。
1. 「映画の日」に映画を見るとき
2. 水曜日に女性が割引を受けるとき
3. 夫婦のどちらかが50歳以上で2人2,000円の割引を利用するとき
4. 毛布やクッションを借りるとき
> **正解：3**　「ただし、チケットを買う際は、身分証明書の提示が必要です」は、直前の「夫婦のどちらかが50歳以上」の割引について述べている。年齢を証明する必要があるため。
