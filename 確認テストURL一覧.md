## テストを更新した場合

HTMLファイルを更新したら、以下のコマンドでGitHub Pagesにも反映できます：
```bash
cd /Users/hayashi./datax/Business/school/jlpt-kakunin-test-deploy
cp /Users/hayashi./datax/Business/school/JLPT/N*/2-kakunin-test/*.html .
git add . && git commit -m "テスト更新" && git push
```
（初回は `git clone https://github.com/takutosquare00-max/jlpt-kakunin-test.git jlpt-kakunin-test-deploy` でクローン）
