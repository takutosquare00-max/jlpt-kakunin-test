# Google Calendar → Google Sheets（読み込み専用）

複数の Google カレンダーからイベントを**読み込んで**スプレッドシートに一覧表示する Google Apps Script です。  
**シートの内容をカレンダーに反映する機能はありません**（編集トリガー・同期メニューなし）。

> **詳細な適用手順**は [適用手順.md](./適用手順.md) を参照してください。

## 列の意味

| 列 | 項目 | 説明 |
|---|------|------|
| A | イベントID | Google カレンダー上のID（参照用） |
| B | カレンダーID | そのイベントが属するカレンダーのID |
| C | カレンダー名 | カレンダー一覧の表示名 |
| D | タイトル | イベント名 |
| E | 日付 | `yyyy-MM-dd` |
| F | 開始時間 | `HH:mm` または 終日 |
| G | 終了時間 | `HH:mm` または 終日 |
| H | ビデオ(Meet) | Meet の URL（あれば読み込み時に表示） |
| I | 通知(分) | リマインダー（分、カンマ区切り） |
| J | 説明 | 説明文 |

## セットアップの要点

1. スプレッドシートに Apps Script でこの `Code.gs` をコピー
2. **高度な Google サービス**で **Google Calendar API**（識別子 `Calendar`）を有効化
3. `appsscript.json` の `oauthScopes` をこのリポジトリのものに合わせる（初回はエディタのマニフェスト表示をオン）
4. `Code.gs` 先頭の `CONFIG.CALENDAR_IDS` に、取得したいカレンダーを列挙

### 複数カレンダーを指定する例

```javascript
const CONFIG = {
  SHEET_NAME: 'カレンダー連携',
  CALENDAR_IDS: [
    'primary',
    'xxxxxxxxxxxxxxxxxxxxxxxxxx@group.calendar.google.com'
  ],
  SYNC_DAYS_PAST: 30,
  SYNC_DAYS_FUTURE: 90,
  ...
};
```

- `'primary'` … 実行アカウントの**デフォルト（メイン）カレンダー**
- それ以外 … Google カレンダー設定の「カレンダーの統合」などから控えた**カレンダーID**

## メニュー

- **初期セットアップ** … シート・ヘッダー作成。旧版の編集トリガーがあれば削除
- **カレンダーから読み込み** … 過去 `SYNC_DAYS_PAST` 日〜未来 `SYNC_DAYS_FUTURE` 日を全対象カレンダーから取得し、開始時刻でソートして上書き表示
- **カレンダーから読み込み（期間を指定）** … 日付範囲を指定して同様に取得

## 動作の注意

- イベント件数が多いカレンダーは **ページネーション**で取り切ります（1カレンダーあたり大量イベントに対応）
- 実行ユーザーが**アクセスできる**カレンダーIDのみ指定してください（オーナーなら通常問題ありません）
- 以前の「双方向連携」版から更新した場合は **初期セットアップ**を再度実行し、ヘッダー行を最新にしてください
