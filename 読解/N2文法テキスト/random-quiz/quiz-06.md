# N2文法 ランダムテスト No.06（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　B：パソコンが使えなかったら何もできないんだから、できないと（　　）からね。
1. 言うというものではない　　2. 言いかねない　　3. 言わざるを得ない　　4. 言ってはいられない

**問2**　この地域は水不足で、砂漠化が進む（　　）。
1. ことはない　　2. というものだ　　3. 一方だ　　4. わけにはいかない

**問3**　パンダは ＿＿ ＿＿ ★ ＿＿ 一面がある。
1. 凶暴な　　2. に反して　　3. イメージ　　4. 外見の

**問4**　この ＿＿ ＿＿ ★ ＿＿ 簡単に作れます。
1. 鍋さえ　　2. 料理でも　　3. どんな　　4. あれば

**問5**　家賃や人件費など、売り上げの多い少ない（　　）毎月費用がかかることを考えなければならない。
1. に応じて　　2. に限り　　3. にかかわらず　　4. につき

**問6**　甘いものはつい食べすぎてしまい（　　）ですが、健康を考えるとあまり食べないほうがいいでしょう。
1. がち　　2. げ　　3. しだい　　4. たい

**問7**　ABKハウジングは東京（　　）、関東地方６県の不動産を取り扱っております。
1. くらい　　2. を目的に　　3. をはじめ　　4. のもとで

**問8**　B：そうじゃないのよ。＿＿ ＿＿ ★ ＿＿ 好きなのよ。
1. 雰囲気が　　2. というより　　3. 好き　　4. お酒が

**問9**　こちらの商品はセール品でございますので、交換はご遠慮（　　）。
1. 願います　　2. 存じます　　3. いたしません　　4. うけたまわりません

**問10**　若年化する青少年犯罪の防止策は ＿＿ ＿＿ ★ ＿＿ 問題だ。
1. ばかりでなく　　2. 学校　　3. 社会全体で　　4. 考えるべき

---

## 解答と解説

**問1**　正解：**4**（言ってはいられない）
「言わざるを得ない」＝やむを得ず言わなければならない。「できないと言わざるを得ない」は、できないと正直に言うしかないという意味。「かねない」は〜してしまう可能性がある場合に使う。

**問2**　正解：**3**（一方だ）
「〜一方だ」＝〜がどんどん進んでいる（悪化・変化が続く）。「砂漠化が進む一方だ」は砂漠化がどんどん進んでいるという意味。「ことはない」は〜する必要はない（義務の否定）。

**問3**　正解：**2**（に反して）
正しい語順：「外見の→イメージ→★に反して→凶暴な」。完成文：「パンダは外見のイメージに反して凶暴な一面がある。」「〜に反して」＝〜と反対に・〜に逆らって。

**問4**　正解：**3**（どんな）
正しい語順：「鍋さえ→あれば→★どんな→料理でも」。完成文：「この鍋さえあれば、どんな料理でも簡単に作れます。」「〜さえ〜ば」＝〜という条件だけで十分。

**問5**　正解：**3**（にかかわらず）
「にかかわらず」＝〜に関係なく。「多い少ないにかかわらず」は売上の多少にかかわらずという意味。「に応じて」は程度に合わせる、「に限り」は〜だけという意味。

**問6**　正解：**1**（がち）
「〜がち」＝〜しやすい・〜になりがち（よくない傾向）。「食べすぎてしまいがち」はついつい食べすぎてしまう傾向があるという意味。「〜げ」は〜そうな様子、「〜しだい」は〜次第で。

**問7**　正解：**3**（をはじめ）
「〜をはじめ」＝〜を代表として（他にも）。「東京をはじめ」は東京を筆頭に関東各県も、という意味。「のもとで」は〜の下で、「を目的に」は目的を表す。

**問8**　正解：**2**（というより）
正しい語順：「お酒が→好き→★というより→雰囲気が」。完成文：「お酒が好きというより雰囲気が好きなのよ。」「〜というより」＝〜ではなくむしろ〜（より正確な表現に言い換える）。

**問9**　正解：**1**（願います）
「ご遠慮願います」＝丁寧に断る表現。「ご〜願います」は相手に〜をお願いするという敬語表現。「ご遠慮願います」は「控えてください」を丁寧に言う表現。

**問10**　正解：**3**（社会全体で）
正しい語順：「学校→ばかりでなく→★社会全体で→考えるべき」。完成文：「学校ばかりでなく、社会全体で考えるべき問題だ。」「〜ばかりでなく」＝〜だけでなく（範囲を広げる）。
