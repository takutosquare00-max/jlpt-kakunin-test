# N2文法 ランダムテスト No.13（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　性能がいいと言われても、実際に（　　）ことには本当にいいかどうかわからない。
1. 売ってみる　　2. 売ってみない　　3. 使ってみる　　4. 使ってみない

**問2**　練習の苦しさも時間が ＿＿ ＿＿ ★ ＿＿ 変わっていった。
1. 楽しい　　2. 思い出に　　3. たつ　　4. につれて

**問3**　彼は、自分のミスで仕事が遅れたのに、あれこれ言い訳した（　　）、結局一言も謝らなかった。
1. にしては　　2. からには　　3. ことなく　　4. あげく

**問4**　怖くて眠れなくなる（　　）、ホラー映画なんて見なければいいのに。
1. のなんのって　　2. どころか　　3. くらいなら　　4. 反面

**問5**　この温泉は、＿＿ ＿＿ ★ ＿＿ 効果もあります。
1. 回復　　2. 疲労　　3. 美肌の　　4. に加えて

**問6**　この製品の使用法（　　）は、ホームページをご覧ください。
1. について　　2. にあたって　　3. をめぐって　　4. に基づいて

**問7**　A：山本先生、今度の日曜日はお宅に（　　）か。
1. おこしになります　　2. おいでになります　　3. お見えになります　　4. おります

**問8**　本場のタイ料理を作ってほしいと頼まれたが、材料がないので（　　）。
1. 作るおそれがある　　2. 作るわけではない　　3. 作りようがない　　4. 作るものだ

**問9**　ペンギンは子どものときは灰色ですが、成長する（　　）黒くなります。
1. にしたがって　　2. どころか　　3. ものの　　4. だけ

**問10**　特別展を（　　）皆様は入館前に入口の注意事項をお読みください。
1. ご覧になる　　2. ご覧に入れる　　3. 拝見する　　4. 拝見なさる

---

## 解答と解説

**問1**　正解：**4**（使ってみない）
「〜ないことには」＝〜しなければ（条件）。「使ってみないことにはわからない」は実際に使わなければわからないという意味。「ないことには」は否定条件を表す重要な表現。

**問2**　正解：**1**（楽しい）
正しい語順：「たつ→につれて→★楽しい→思い出に」。完成文：「練習の苦しさも時間がたつにつれて、楽しい思い出に変わっていった。」「〜につれて」＝〜が変化するのに伴って（同時変化）。

**問3**　正解：**4**（あげく）
「〜あげく」＝さんざん〜した末に（よくない結果）。「言い訳したあげく謝らなかった」は、散々言い訳をして最終的にも謝らなかったという意味。「からには」は〜した以上は（決意・義務）を表す。

**問4**　正解：**3**（くらいなら）
「〜くらいなら〜ほうがいい」＝〜するくらいだったら別の方がいい。「眠れなくなるくらいなら見なければいい」はそんなに怖いならやめればいいのに、という意味。

**問5**　正解：**4**（に加えて）
正しい語順：「疲労→回復→★に加えて→美肌の」。完成文：「この温泉は、疲労回復に加えて美肌の効果もあります。」「〜に加えて」＝〜のほかに・それに加えて（追加情報）。

**問6**　正解：**1**（について）
「〜について」＝〜に関して（テーマ・話題）。「使用法について」は使用法に関する情報という意味。「をめぐって」は〜を中心に意見が分かれる場合に使う。「に基づいて」は〜を根拠にして。

**問7**　正解：**2**（おいでになります）
「おいでになる」＝「いる・来る・行く」の尊敬語。先生（目上の人）がいるかを聞いているので尊敬語。「おります」は「いる」の謙譲語（自分が使う）。「おこしになる」「お見えになる」は来るの意味が強い。

**問8**　正解：**3**（作りようがない）
「〜ようがない」＝どうしても〜できない（方法・手段がない）。「作りようがない」は材料がないので作ることが不可能という意味。「〜わけではない」は必ずしも〜ではない、という婉曲否定。

**問9**　正解：**1**（にしたがって）
「〜にしたがって」＝〜の変化・進行に合わせて（同時変化）。「成長するにしたがって黒くなる」は成長に伴って色が変わるという意味。「につれて」と同じ意味で使える。

**問10**　正解：**1**（ご覧になる）
「ご覧になる」＝「見る」の尊敬語（相手・第三者の行為）。お客様が見る行為なので尊敬語を使う。「拝見する」は「見る」の謙譲語（自分の行為）。「ご覧に入れる」は見せるの謙譲語。
