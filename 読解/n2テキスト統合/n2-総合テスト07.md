## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　日本製品は国際的な価格 <u>競争</u> の中で苦境に立たされている。
1. きょうそう　　2. きょうぞう　　3. けいそう　　4. きょそう
> **正解：1（きょうそう）**　「競争」＝勝ち負けを争うこと。「競」＝キョウ、「争」＝ソウ。「きょうそう（競走）」と同音だが、価格や市場の場合は「競争」を使う。

**2**　日本のサービスは、海外でも高く <u>評価</u> されている。
1. ひょうか　　2. ひょうけ　　3. へいか　　4. ひゅうか
> **正解：1（ひょうか）**　「評価」＝物事の価値を判断すること。「評」＝ヒョウ、「価」＝カ。「へいか（平価）」などと混同しないよう注意。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　この警備会社は、セキュリティーシステムに（　　）がないか常にチェックしている。
1. 問題　　2. 課題　　3. 話題　　4. 議題
> **正解：1（問題）**　「問題がないかチェックする」＝不具合や故障がないか確認する。「課題」は解決すべきテーマ、「話題」は話の種、「議題」は会議で話し合うことがら。

**4**　料金は少し高くても、頼れる会社に警備を（　　）たいというニーズがある。
1. 任せ　　2. 預け　　3. 届け　　4. 求め
> **正解：1（任せ）**　「（仕事を）任せる」＝相手にゆだねて担当してもらう。「警備を任せる」のように使う。「預ける」は物や人を一時的にゆだねる場合に使う。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　卒業論文が間に合わなかった（　　）、卒業が半年遅れてしまった。
1. ばかりに　　2. からには　　3. ものの　　4. にもかかわらず
> **正解：1（ばかりに）**　「〜ばかりに」＝〜という一つの理由だけで（後悔・残念な結果）。「間に合わなかったばかりに」はそれだけが原因で大変なことになったという意味。「にもかかわらず」は〜なのに（逆接）。

**6**　A:加藤君のお父さん、優しそうだよね。　B:優しい（　　）。いつも怒ってばかりいるよ。
1. ことか　　2. ことだ　　3. もんか　　4. ものだ
> **正解：3（もんか）**　「〜もんか」＝〜なわけがない（強い否定・反語）。「優しいもんか」はちっとも優しくないという強い否定。「ことか」は〜ことだろうか（感嘆）、「ものだ」は一般的な傾向・感慨。

**7**　A:来週の同窓会、行くでしょ？　B:ごめん。ゼミの合宿があって、休む（　　）んだ。
1. わけではない　　2. ものではない　　3. わけにはいかない　　4. ことはない
> **正解：3（わけにはいかない）**　「〜わけにはいかない」＝〜することはできない（社会的・道義的な理由）。「休むわけにはいかない」は合宿があるので休んではいけないという意味。「わけではない」は必ずしも〜ではない。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　注意書きを ＿＿ ＿＿ <span class="star">★</span> ＿＿ 受け付けてもらえなかった。
1. 書類が　　2. ばかりに　　3. 見落としていた　　4. 足りなくて
> **正解：1（書類が）**　正しい語順：「見落としていた→ばかりに→★書類が→足りなくて」。完成文：「注意書きを見落としていたばかりに、書類が足りなくて受け付けてもらえなかった。」「〜ばかりに」＝〜だけが原因で（後悔）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　日本製品は国際的な価格競争の中で苦境に立たされている。しかし競争できないとあきらめるわけにはいかないし、悲観してはいられない。それでは日本のビジネスが世界的に発展する道はどこにあるのだろうか。ビジネス成功のポイントは、ただ消費者のニーズを追求することのみだ。今、ヨーロッパでは日本の警備会社が人気を集めている。セキュリティーシステムに問題がないか常にチェックし、警報器が鳴ったら、すぐ担当者が駆けつけ、状況を把握して客に連絡する。日本では当たり前のこのサービスがヨーロッパで高く評価されているという。「セキュリティーがしっかりしていなかったばかりに、大きな被害に遭った」と後悔するより、料金は少し高くても、頼れる会社に任せようというニーズがあったと言える。日本では当然のことだが海外では高く評価される、それこそが国際競争力を持つ商品というものだ。

**9**　ヨーロッパで日本の警備会社が人気を集めているのはなぜか。
1. セキュリティーシステムをチェックしてくれるから
2. いつでもチェックしていて、すぐ問題に対応してくれるから
3. 料金が少し高いから
4. 安心して警備を任せられないから
> **正解：2**　「常にチェックし、警報器が鳴ったら、すぐ担当者が駆けつけ…連絡する」とある。常時の監視と迅速な対応が評価されている。

**10**　この文章で筆者が最も言いたいことは何か。
1. 国際的な経済競争が激しくなっていること
2. 日本製品が外国製品に比べて、値段が高くなったこと
3. 日本の警備会社がヨーロッパで人気を集めていること
4. 消費者のニーズを追求することが、世界市場で勝つということ
> **正解：4**　筆者は「ビジネス成功のポイントは、ただ消費者のニーズを追求することのみだ」と述べ、警備会社を例にそれを説明している。最も伝えたいのはニーズの追求が国際競争力につながるという点。
