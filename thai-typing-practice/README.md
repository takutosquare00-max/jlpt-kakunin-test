# タイピングで覚えるタイ文字の読み

タイ文字を見て、ラテン文字の発音を入力する練習アプリです。iPhone・タブレットでの利用を想定した静的HTMLアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. 練習レベルを選ぶ
3. 表示されたタイ文字の読みをラテン文字で入力
4. 正解すると自動で次の問題に進む

## 練習レベル

- 文字: タイ子音44字と基本母音・特殊母音32項目
- 音節: 母音体系、末子音、声調記号を組み合わせた生成ドリル
- 単語: 初級の基本語彙
- 短文: 初級の短い文
- すべて: 全問題を混ぜて練習

## 収録内容

- 文字: 76問
- 音節: 1468問
- 単語: 84問
- 短文: 20問
- 合計: 1648問

音節は全単語辞書ではなく、タイ文字の読み方を体系的に反復するための生成ドリルです。

## データ形式

問題データは `index.html` 内で生成しています。追加で固定問題を入れる場合は、`thai-data` JSONに次の形式で追加できます。

```json
{
  "thai": "บ้าน",
  "reading": "baan",
  "answers": ["baan", "ban"],
  "meaning_ja": "家",
  "level": "word",
  "category": "生活",
  "hint": "บ + า + ้ + น",
  "tone": "falling"
}
```
