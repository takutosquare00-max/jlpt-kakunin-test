## 練習B

**117** 1. 例：→ スーパーへ　行きます。
- 1) →
- 2) →
- 3) →
- 4) →

（絵：例＝スーパー、1）郵便局、2）デパート、3）銀行、4）美術館）

**118** 2. 例：けさ → けさ　どこへ　行きましたか。……スーパーへ　行きました。
- 1) 先月 →
- 2) きのうの　午後 →
- 3) 来週の　月曜日 →
- 4) 先週の　日曜日 →

（絵：例＝スーパー、1）アメリカ、2）図書館、3）パワーでんき、4）×（どこも行かなかった））

**119** 3. 例：→ 何で　京都へ　行きますか。……電車で　行きます。
- 1) →
- 2) →
- 3) →
- 4) →

（絵：例＝電車/京都、1）自転車/学校、2）飛行機/東京、3）船/九州、4）徒歩/駅）

**120** 4. 例：京都（友達）→ だれと　京都へ　行きますか。……友達と　行きます。
- 1) 美術館（彼女）→
- 2) 広島（会社の人）→
- 3) 北海道（家族）→
- 4) フランス（一人で）→

**121** 5.
- 例1：京都（3月3日）→ いつ　京都へ　行きますか。……3月3日に　行きます。
- 例2：東京（来週）→ いつ、東京へ　行きますか。……来週　行きます。
  1. さくら大学（9月14日）→
  2. アメリカ（来年の3月）→
  3. 広島（来月）→
  4. 病院（今週の水曜日）→

**122** 6. 例：いつ　日本へ　来ましたか。→ 去年の9月に　来ました。
- 1) だれと　日本へ　来ましたか。→
- 2) 先月　どこへ　行きましたか。→
- 3) 何で　韓国へ　行きましたか。→
- 4) いつ　国へ　帰りますか。→

（絵：先月＝韓国、去年9月＝日本、来年12月＝帰国）

**123** 7. 例：毎朝　何時に　会社へ　行きますか。→ 7時半に　行きます。
- 1) 毎朝　何で　会社へ　行きますか。→
- 2) 毎晩　何時に　うちへ　帰りますか。→
- 3) おととい　だれと　大阪城へ　行きましたか。→
- 4) 土曜日　どこへ　行きますか。→

（カレンダー：きょう＝水曜日　月曜日＝7:30出発/電車で会社　火曜日＝8:00帰宅　土曜日＝休み）

**124** 8. 例：→ ミラーさんの　誕生日は　いつですか。……10月6日です。
- 1) →（山田：3月27日）
- 2) →（カリナ：4月10日）
- 3) →（サントス：11月9日）
- 4) →（マリア：9月1日）

## 練習C

**125** 1.
- A：あしたは　日曜日ですね。
- B：あ、そうですね。
- A：わたしは　大阪城へ　行きます。タワポンさんは？
- B：どこも　行きません。勉強します。

練習：
1. （絵：美術館へ行く人、家で休む人）
2. （絵：京都へ行く人、自転車の人）

**126** 2.
- A：先週　東京へ　行きました。これ、お土産です。どうぞ。
- B：ありがとう　ございます。一人で　行きましたか。
- A：いいえ、友達と　行きました。
- B：何で　行きましたか。
- A：バスで　行きました。

練習：
1. （絵：車で友達と東京へ）
2. （絵：飛行機で一人で東京へ）

**127** 3.
- A：お国は　どちらですか。
- B：アメリカです。
- A：そうですか。いつ　日本へ　来ましたか。
- B：去年の　9月に　来ました。

練習：
1. （絵：インドネシア、先月来日）
2. （絵：タイ、先週来日）
