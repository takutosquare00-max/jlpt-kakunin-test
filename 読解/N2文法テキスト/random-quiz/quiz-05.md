# N2文法 ランダムテスト No.05（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　お客様が ＿＿ ＿＿ ★ ＿＿ ください。
1. 2階の　　2. お見えになったら　　3. ご案内して　　4. 応接室に

**問2**　B：優しい（　　）。いつも怒ってばかりいるよ。
1. ことか　　2. ことだ　　3. もんか　　4. ものだ

**問3**　卒業式はこのプログラム（　　）行いますので、よろしくお願いします。
1. に先立って　　2. とともに　　3. にそって　　4. に関して

**問4**　自分がされて嫌な ＿＿ ＿＿ ★ ＿＿ ではない。
1. もの　　2. ことを　　3. する　　4. 他人に

**問5**　A：この川、以前に（　　）、ずいぶんきれいになりましたね。
1. 応じて　　2. つれて　　3. わたって　　4. 比べて

**問6**　当学会は、研究者や専門家（　　）、企業や個人の方々にも広く開かれた学会です。
1. に反して　　2. に比べて　　3. のみならず　　4. にもかかわらず

**問7**　彼の理想はわかるが、あまりに現実離れしているので、賛成し（　　）。
1. ぬいた　　2. がたい　　3. がちだ　　4. かねない

**問8**　ABK社は一流企業（　　）給料が安くてびっくりした。
1. にしては　　2. はもとより　　3. を問わず　　4. さえ

**問9**　水泳は子どもからお年寄りまで世代（　　）楽しめるスポーツだと言われている。
1. ばかりでなく　　2. に限って　　3. を問わず　　4. とおりに

**問10**　これは苦労して探し回った（　　）、やっと見つけた本なんです。
1. あげく　　2. あまり　　3. 末に　　4. 限り

---

## 解答と解説

**問1**　正解：**4**（応接室に）
正しい語順：「お見えになったら→2階の→★応接室に→ご案内して」。完成文：「お客様がお見えになったら、2階の応接室にご案内してください。」「お見えになる」＝来るの尊敬語。

**問2**　正解：**3**（もんか）
「〜もんか」＝〜なわけがない（強い否定・反語）。「優しいもんか」はちっとも優しくないという強い否定。「ことか」は〜ことだろうか（感嘆）、「ものだ」は一般的な傾向・感慨を表す。

**問3**　正解：**3**（にそって）
「〜にそって」＝〜に従って・〜に沿って。「プログラムにそって行う」はプログラムの通りに進めるという意味。「に先立って」は〜の前に、「に関して」は〜についてのテーマを表す。

**問4**　正解：**3**（する）
正しい語順：「ことを→他人に→★する→もの」。完成文：「自分がされて嫌なことを他人にするものではない。」「〜するものではない」＝〜すべきでない（道徳的な否定・教訓）。

**問5**　正解：**4**（比べて）
「〜に比べて」＝〜と比較して。「以前に比べてきれいになった」は前と比べると改善されたという意味。「につれて」は変化の並行進行、「にわたって」は範囲の広がりを表す。

**問6**　正解：**3**（のみならず）
「〜のみならず」＝〜だけでなく（書き言葉）。「研究者のみならず企業にも」は研究者だけでなく企業にも開かれているという意味。「ばかりでなく」と同じ意味だが、より改まった表現。

**問7**　正解：**2**（がたい）
「〜がたい」＝〜しにくい・〜できない（困難・不可能）。「賛成しがたい」は賛成することが難しいという意味。「かねない」は〜してしまう恐れがある（よくない可能性）、「がちだ」は〜しやすい（悪い傾向）。

**問8**　正解：**1**（にしては）
「〜にしては」＝〜の割には（期待外れ・予想と異なる）。「一流企業にしては給料が安い」は一流企業のわりに安かったという意外感を表す。「はもとより」は〜はもちろん、を表す。

**問9**　正解：**3**（を問わず）
「を問わず」＝〜に関係なく、どんな〜でも。「世代を問わず」はどの世代でも関係なくという意味。「に限って」は〜だけ、「ばかりでなく」は〜だけでなくという意味。

**問10**　正解：**3**（末に）
「〜末に」＝長い過程・苦労の後に（努力の結果）。「探し回った末にやっと見つけた」は苦労して最終的に見つけたという意味。「あげく」は〜した末に（よくない結果）、「あまり」は〜すぎて（感情の度合い）。
