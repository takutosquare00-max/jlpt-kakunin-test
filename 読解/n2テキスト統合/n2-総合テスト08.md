## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　発掘によって、古代文明の <u>謎</u> が少しずつ明らかになってきた。
1. なぞ　　2. めい　　3. なし　　4. なん
> **正解：1（なぞ）**　「謎」＝まだ解明されていない不思議なこと。訓読みで「なぞ」。「謎を解く」「宇宙の謎」のように使う。

**2**　科学 <u>技術</u> の進歩によって、多くのことがわかるようになった。
1. ぎじゅつ　　2. きじゅつ　　3. ぎじゅっ　　4. ぎじつ
> **正解：1（ぎじゅつ）**　「技術」＝物事を行うわざ・テクノロジー。「技」＝ギ、「術」＝ジュツ。「きじゅつ（記述）」と読み間違えやすいので注意。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　天体（　　）の技術が進んだことで、ブラックホールの存在がわかった。
1. 観測　　2. 観察　　3. 観光　　4. 観賞
> **正解：1（観測）**　「観測」＝自然現象を機器で測り調べること。「天体観測」「気象観測」のように使う。「観察」は注意して見ること、「観賞」は美しいものを味わうこと。

**4**　研究が進むほど、宇宙の謎はかえって（　　）ばかりだ。
1. 深まる　　2. 広がる　　3. 高まる　　4. 強まる
> **正解：1（深まる）**　「謎が深まる」＝なぞがいっそうわからなくなる。「深まる」は程度が深くなる意味で「謎」と結びつく。「広がる」は範囲、「高まる」は度合いが上がること。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　今回のマラソンは大方の予想（　　）、初出場の無名のランナーが優勝した。
1. に関して　　2. に反して　　3. のみならず　　4. からいうと
> **正解：2（に反して）**　「〜に反して」＝〜に反対の結果・〜と違って。「予想に反して」は予想とは逆の結果だったという意味。

**6**　当学会は、研究者や専門家（　　）、企業や個人の方々にも広く開かれた学会です。
1. に反して　　2. に比べて　　3. のみならず　　4. にもかかわらず
> **正解：3（のみならず）**　「〜のみならず」＝〜だけでなく（書き言葉）。「研究者のみならず企業にも」は研究者だけでなく企業にも開かれているという意味。

**7**　グレープフルーツは木になっている様子がブドウに似ている（　　）、その名前がついたそうだ。
1. のみならず　　2. ことから　　3. にもかかわらず　　4. ものの
> **正解：2（ことから）**　「〜ことから」＝〜という理由・根拠から。「ブドウに似ていることから名前がついた」は似ているという事実が名前の由来になったという意味。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　パンダは ＿＿ ＿＿ <span class="star">★</span> ＿＿ 一面がある。
1. 凶暴な　　2. に反して　　3. イメージ　　4. 外見の
> **正解：2（に反して）**　正しい語順：「外見の→イメージ→★に反して→凶暴な」。完成文：「パンダは外見のイメージに反して凶暴な一面がある。」「〜に反して」＝〜と反対に・〜に逆らって。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　科学技術が進み、知識という側面から見ると、昔よりはるかに多くのことがわかるようになった。宇宙に関しても様々な事実が明らかになってきた。例えば、昔はブラックホールの存在さえわからなかったが、天体観測などの技術が進んだことから、その存在がわかった。さらに、調査が進むにしたがって、ブラックホールはただそこに存在するのみならず、膨張していることも、明らかになってきた。しかし、様々な事実が明らかになる一方で、ブラックホールとは何なのか、なぜ存在するのか、膨張し続けたらどうなるのか、さらなる疑問がわいてくる。これまでは知識を得ることによって、すべてがわかると期待されていた。しかし科学者たちが日々研究を続けているにもかかわらず、宇宙の謎は深まるばかりである。

**9**　科学技術が進んだ結果、ブラックホールについて何がわかったか。
1. 存在することと、膨張していること
2. もともと存在しないということ
3. 色が少しずつ変わっていくこと
4. やがて消えてなくなること
> **正解：1**　「その存在がわかった」「ただそこに存在するのみならず、膨張していることも、明らかになってきた」とある。存在と膨張の二点が判明した。

**10**　この文章で筆者が最も言いたいことは何か。
1. 知識を得れば、いつかすべてがわかるということ
2. 科学者は研究をやめるべきだということ
3. 事実が明らかになるほど新たな疑問が生まれ、宇宙の謎はかえって深まること
4. 宇宙の研究は意味がないということ
> **正解：3**　「様々な事実が明らかになる一方で、さらなる疑問がわいてくる」「宇宙の謎は深まるばかりである」とある。わかるほど新たな謎が増えるという点が主張。
