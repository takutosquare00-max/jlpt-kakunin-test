# 解答一覧

## パート1 解答

| 問 | 正解 | 文法ポイント | 課 |
|---|---|---|---|
| 問1 | **b. に** | 〜にいただきます：人＋に＋もの＋を＋いただく（謙譲の授受） | 41 |
| 問2 | **c. が** | 〜がくださいました：目上の人が主語で〜くれるの尊敬 | 41 |
| 問3 | **a. やりました** | 〜てやります：目下・動植物への行為の授受 | 41 |
| 問4 | **b. くださいません**（a/d も正解扱い） | 〜てくださいませんか：丁寧な依頼。〜ていただけませんか／〜てもらえませんか も依頼として可。 | 41 |
| 問5 | **a. のために** | 名詞＋のために：目的（健康のために） | 42 |
| 問6 | **b. をするのに** | 動詞辞書形＋のに：用途・手段（何をするのに使う） | 42 |
| 問7 | **b. ように** | 動詞可能形＋ように：能力獲得の目的（無意志動詞） | 42 |
| 問8 | **b. によって** | 名詞＋によって：多様性（〜によって異なる） | 42 |
| 問9 | **d. 出し** | 動詞ます形＋そう：様態（雨が降りそう／降り出しそう） | 43 |
| 問10 | **b. そう** | い形容詞語幹＋そう：様態（おいしそう＝〜く見える） | 43 |
| 問11 | **a. なって** | 〜くなってくる：変化が話者に近づく（寒くなってきた） | 43 |
| 問12 | **b. くれません** | 〜てくれませんか：くだけた依頼表現 | 43 |
| 問13 | **a. すぎて** | い形容詞語幹＋すぎる：過度（難しすぎる） | 44 |
| 問14 | **b. やすい**（c も正解扱い） | 動詞ます形＋やすい／にくい：細くて書きやすい／書きにくい どちらも自然 | 44 |
| 問15 | **b. 静かに** | な形容詞＋に＋する：状態への変化（静かにする） | 44 |
| 問16 | **a. 場合は** | 名詞＋の＋場合は：仮定の場面設定（雨の場合は） | 45 |
| 問17 | **c. のに** | 〜のに：逆接（〜なのに予想と違う結果） | 45 |
| 問18 | **a. の** | 暖房の温度：助詞の（連体修飾） | 41 |

## 文法まとめ（Unit41〜45）

| 課 | 文法項目 | 形 | 例 |
|---|---|---|---|
| 41 | いただく | 〜に＋もの＋を＋いただく | 先生にお土産をいただきました |
| 41 | くださる | 〜が＋[わたしに]＋もの＋を＋くださる | 部長が本をくださいました |
| 41 | やる | 〜に＋もの＋を＋やる（目下・動植物） | 犬にえさをやる／息子に本をやる |
| 41 | 〜ていただく | 動詞て形＋いただく | 駅まで送っていただきました |
| 41 | 〜てくださる | 動詞て形＋くださる | 先生が説明してくださいました |
| 41 | 〜てやる | 動詞て形＋やる | 子どもに絵本を読んでやる |
| 41 | 〜てくださいませんか | 動詞て形＋くださいませんか | ちょっと教えてくださいませんか |
| 42 | 名詞のために | 名詞＋のために | 家族のために働く |
| 42 | 動詞辞書形＋ために | 動詞辞書形＋ために | 留学するためにお金をためる |
| 42 | 動詞辞書形／可能形＋ように | 意志動詞は ために、無意志・可能・否定は ように | 忘れないように／話せるように |
| 42 | 動詞辞書形＋のに | 用途・手段（〜のに使う／要る） | はさみは紙を切るのに使う |
| 42 | 〜は／〜も／〜によって | 数量・多様性 | 1万人は来た／国によってちがう |
| 43 | 動詞ます形＋そう | 様態（直接的な観察） | 雨が降りそう・落ちそう |
| 43 | い形容詞語幹＋そう／な形容詞＋そう | 様態（見た目の印象） | おいしそう／元気そう |
| 43 | 〜てきます | 変化・継続・移動（話者へ近づく） | 寒くなってきた／買ってきます |
| 43 | 〜ていきます | 変化・継続・移動（話者から離れる） | これから増えていく／歩いていく |
| 43 | 〜てくれませんか | 動詞て形＋くれませんか | ちょっと手伝ってくれませんか |
| 44 | 〜すぎます | 動詞ます形／い形容詞語幹／な形容詞＋すぎる | 食べすぎる・高すぎる・静かすぎる |
| 44 | 〜やすい／〜にくい | 動詞ます形＋やすい／にくい | 書きやすい／飲みにくい |
| 44 | 〜く／に する／なる | い形容詞＋く／な形容詞＋に＋する／なる | 大きくする／静かになる |
| 44 | 〜にします | 〜にします（選択・決定） | コーヒーにします |
| 45 | 〜場合は | 名詞の／動詞普通形＋場合は | 雨の場合は／遅れる場合は |
| 45 | 〜のに | 普通形＋のに（逆接） | 高いのに買った／勉強したのに落ちた |
| 45 | 〜のに／〜が／〜ても | 逆接の使い分け | のに：意外感／が：単純対比／ても：仮定逆接 |
