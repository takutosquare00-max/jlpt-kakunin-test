# N2文法 ランダムテスト No.09（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　人は自分の経験（　　）判断することが多いので、正しい判断をするにはたくさんの経験が必要だと言われる。
1. にしては　　2. に基づいて　　3. に加えて　　4. にあたり

**問2**　結婚しない若者の増加も問題だが、＿＿ ＿＿ ★ ＿＿ 人が増えているのも問題だろう。
1. 子どもを産もう　　2. 結婚した　　3. としない　　4. としても

**問3**　彼は他人に ＿＿ ＿＿ ★ ＿＿ ので、みんなに嫌がられている。
1. 甘い　　2. 厳しい　　3. 自分に　　4. わりに

**問4**　原子力発電所の安全性（　　）、世界各地で議論が行われている。
1. を込めて　　2. を問わず　　3. をめぐって　　4. をもとに

**問5**　卒業論文が間に合わなかった（　　）、卒業が半年遅れてしまった。
1. ばかりに　　2. からには　　3. ものの　　4. にもかかわらず

**問6**　B：まだ時間はありますから、急ぐ（　　）よ。
1. どころではありません　　2. わけにはいきません　　3. おそれがあります　　4. ことはありません

**問7**　救助隊が現場に ＿＿ ＿＿ ★ ＿＿ ことになっています。
1. 開始する　　2. 救援活動を　　3. 着き　　4. 次第

**問8**　プレッシャーで実力が出せなかったとき、自分に負けた気がして、（　　）。
1. くやしいようになっている　　2. くやしかったつもりだ　　3. くやしくてならなかった　　4. くやしいおそれがあった

**問9**　高橋さんは子どもが生まれたの（　　）、たばこをやめる決心をしたんだそうです。
1. はもとより　　2. をきっかけに　　3. をはじめとして　　4. を中心として

**問10**　歴史ある建物だが、古くなって危険なので（　　）。
1. 壊してしょうがない　　2. 壊さないはずだ　　3. 壊しっこない　　4. 壊さざるを得ない

---

## 解答と解説

**問1**　正解：**2**（に基づいて）
「〜に基づいて」＝〜を根拠・基準にして。「経験に基づいて判断する」は経験を根拠にして判断するという意味。「に加えて」は〜のほかに（追加）、「にしては」は〜の割には（期待外れ）。

**問2**　正解：**1**（子どもを産もう）
正しい語順：「結婚した→としても→★子どもを産もう→としない」。完成文：「結婚したとしても子どもを産もうとしない人が増えているのも問題だろう。」「〜としても」＝〜だとしても（譲歩の仮定）。

**問3**　正解：**3**（自分に）
正しい語順：「厳しい→わりに→★自分に→甘い」。完成文：「彼は他人に厳しいわりに自分に甘いので、みんなに嫌がられている。」「〜わりに」＝〜の割には（ギャップ・矛盾）。

**問4**　正解：**3**（をめぐって）
「〜をめぐって」＝〜を話題・争点として（意見が分かれる問題）。「安全性をめぐって議論が行われている」は安全性が問題の中心になっているという意味。「をもとに」は〜を材料・根拠にして。

**問5**　正解：**1**（ばかりに）
「〜ばかりに」＝〜という一つの理由だけで（後悔・残念な結果）。「間に合わなかったばかりに」はそれだけが原因で大変なことになったという意味。「にもかかわらず」は〜なのに（逆接）。

**問6**　正解：**4**（ことはありません）
「〜ことはない」＝〜する必要はない。「急ぐことはありません」は急がなくていいという意味。「わけにはいかない」は〜することはできない（できない理由がある場合）。

**問7**　正解：**2**（救援活動を）
正しい語順：「着き→次第→★救援活動を→開始する」。完成文：「救助隊が現場に着き次第、救援活動を開始することになっています。」「〜次第」＝〜したらすぐに。動詞ます形に接続する点に注意。

**問8**　正解：**3**（くやしくてならなかった）
「〜てならない」＝〜で仕方ない（感情・感覚が強くて抑えられない）。「くやしくてならなかった」はくやしくてたまらなかったという意味。「〜てしかたない」「〜てたまらない」と同じ意味。

**問9**　正解：**2**（をきっかけに）
「〜をきっかけに」＝〜を転機・きっかけとして。「生まれたのをきっかけに」は、その出来事が行動変容のきっかけになったという意味。「はもとより」は〜はもちろん、を表す。

**問10**　正解：**4**（壊さざるを得ない）
「〜ざるを得ない」＝したくないがやむを得ずしなければならない。「壊さざるを得ない」は仕方なく壊すという意味。「〜っこない」は絶対〜できない、「〜ざるを得ない」は否定形＋を得ない。
