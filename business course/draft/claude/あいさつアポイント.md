## タイ人学習者の特徴と注意点

### なぜ日本語の敬語が難しいか

| タイ語の敬語 | 日本語の敬語 |
|------------|------------|
| 語尾に "krap/kha" を付けるだけ | 尊敬語・謙譲語・丁寧語の3種類 |
| 動詞・名詞は変化しない | 動詞・名詞・表現全体が変わる |
| シンプル・一貫したルール | 場面・相手・動作の主体で変化 |

**指導のポイント：** タイ語の敬語は「語尾一言で完結」するシンプルな仕組みのため、日本語の敬語は根本的に構造が異なる。まず「なぜ形が変わるのか」という文化的背景から説明する。

### タイ人学習者がよく間違える点
- 尊敬語と謙譲語の混同（例：お客様に「うかがいます」を使う ✓ vs 「いらっしゃいます」❌）
- 丁寧にしようとして二重敬語になる（例：「お越しになられる」→「お越しになる」が正しい）
- 主語が省略された文脈の読み取り

## ① 日常会話表現とビジネス会話表現の比較・解説（5〜20分）

### 接客・来客対応の比較

| 言いたいこと | 日常会話 | ビジネス表現 |
|------------|----------|------------|
| いらっしゃい | いらっしゃい / こんにちは | いらっしゃいませ |
| ちょっと待って | ちょっと待ってください | 少々お待ちください |
| お待たせ | お待たせしました | お待たせいたしました |
| 別の人を呼ぶ | ちょっと呼んできます | 担当者をお呼びします |
| わかった | わかりました | かしこまりました |
| ごめんなさい | ごめんなさい | 申し訳ございません |
| ありがとう | ありがとうございます | ありがとうございました |

### 予約・日程調整の比較

| 言いたいこと | 日常会話 | ビジネス表現 |
|------------|----------|------------|
| 来週どう？ | 来週はどうですか？ | 来週はいかがでしょうか。 |
| いつがいい？ | いつがいいですか？ | 何時がよろしいですか。 |
| 〇日か〇日は？ | 〇日か〇日はどうですか？ | 〇月〇日か〇日はいかがでしょうか。 |
| じゃあその日で | じゃあその日にします | それでは、〇月〇日にお伺いいたします。 |
| また会おうね | また会いましょう | お会いできるのを楽しみにしております。 |

### 日程変更の比較

| 言いたいこと | 日常会話 | ビジネス表現 |
|------------|----------|------------|
| ごめん、日にち変えたい | ごめんなさい、変えてもいいですか？ | 申し訳ございませんが、日程を変更してもよろしいでしょうか。 |
| 他の日は？ | 他の日はどうですか？ | 他にご都合のよい日はございますか。 |
| じゃあ〇日で | じゃあ〇日にします | では、〇日にご変更いたします。 |

### 接客8大用語（重要！）

ビジネス日本語の接客では「8大用語」と呼ばれる基本フレーズがある。

| 日本語 | English | 使う場面 |
|--------|---------|----------|
| いらっしゃいませ | Welcome | 来客時の第一声（返事不要） |
| 少々お待ちください | Please wait a moment | 対応に時間がかかる時 |
| お待たせいたしました | Thank you for waiting | 待たせた後に必ず言う |
| 担当者をお呼びします | I will call the person in charge | 自分で対応できない時 |
| かしこまりました | Certainly (most polite) | 了解・承諾（最も丁寧） |
| 申し訳ございません | I sincerely apologize | 謝罪（「すみません」より丁寧） |
| 恐れ入ります | Thank you / Excuse me (contextual) | お礼・クッション言葉として |
| ありがとうございました | Thank you very much | 見送り・会話の締め |

**ポイント：** 「かしこまりました」＞「承知いたしました」＞「了解しました」の順に丁寧さが変わる。お客様には必ず「かしこまりました」か「承知いたしました」を使う。

### クッション言葉

依頼・断り・確認をするとき、直接言う前に**クッション言葉**を置くと印象が大きく変わる。

| クッション言葉 | English | 後に続く表現 |
|------------|---------|-------------|
| 恐れ入りますが | Excuse me, but... | お願い・確認事項 |
| 申し訳ございませんが | I am very sorry, but... | 断り・変更のお願い |
| よろしければ | If you don't mind... | 提案・お勧め |
| お手数おかけしますが | Sorry to trouble you, but... | 相手に作業をお願いする時 |

## ② 日常会話表現からビジネス会話表現への変換練習（20〜35分）

### 練習1：変換練習

日常会話をビジネス表現に直しましょう。

1. 「ちょっと待って。」
   → 「（　　　　）お待ちください。」

2. 「わかりました。」
   → 「（　　　　　　　　）。」または「（　　　　　　　　）。」

3. 「来週どう？」
   → 「来週は（　　　　　　　　　）でしょうか。」

4. 「日にち変えていい？」
   → 「日程を変更しても（　　　　　　　　　）でしょうか。」

5. 「ごめんなさい。」
   → 「（　　　　　　　　　）ございません。」

> 答え：1. 少々　2. かしこまりました / 承知いたしました　3. いかが　4. よろしい　5. 申し訳

---

### 練習2：クッション言葉を加える練習

クッション言葉を使って、より丁寧な文に直しましょう。

```
❌「お名前を書いてください。」（直接的すぎる）
✓ 「恐れ入りますが、こちらにお名前をお書きください。」
```

1. 「もう一度言ってください。」
   → 「（　　　　　　　）が、もう一度おっしゃっていただけますか。」

2. 「明日はできません。」
   → 「（　　　　　　　　　　　　　）が、明日はご対応がいたしかねます。」

> 答え：1. 恐れ入ります　2. 申し訳ございません

---

### 練習3：アポイントの穴埋め練習

（　）に適切な言葉を入れましょう。

```
担当：来週、〇日か〇日でしたら、ご都合はいかが（　　）しょうか？
客　：〇日でお願いします。
担当：かしこまり（　　　　）。それでは、〇日〇時にお伺い（　　　　　）。
客　：よろしくお願いします。
担当：お会いできるのを楽し（　　）にしております。当日はよろしくお願い（　　　　　）。
```

> 答え：で / ました / いたします / み / いたします

## ③ 会話シーンを想定したロールプレイ（35〜45分）

### シナリオA：来客対応 + 予約

```
【場面】
お客様が来訪。時間を聞いて、候補日を2つ提示して予約を入れる。

役割：
・Aさん＝受付担当者
・Bさん＝お客様
```

**会話の流れ（参考）**
- B：「来週、担当の方にお会いしたいのですが。」
- A：時間を確認 → 候補日を2つ提示 → 予約を入れる

**使うフレーズ：** いらっしゃいませ ／ いかがでしょうか ／ 何時がよろしいですか ／ かしこまりました ／ お会いできるのを楽しみにしております

---

### シナリオB：日程変更

```
【場面】
担当者の都合が悪くなった。変更を申し出て、新しい日程を提案する。

役割：
・Aさん＝担当者（変更を申し出る側）
・Bさん＝お客様（受け入れる側）
```

**会話例（参考）**

```
担当：申し訳ございませんが、〇日のご予約を変更してもよろしいでしょうか。
客　：はい、大丈夫ですよ。
担当：ありがとうございます。では、〇日の午後はいかがでしょうか。
客　：承知しました。
担当：それでは、〇日〇時にご変更いたします。よろしくお願いいたします。
```

**使うフレーズ：** 申し訳ございませんが ／ よろしければ〇日は ／ かしこまりました

**評価ポイント**
- クッション言葉を使っているか
- 「かしこまりました」「承知いたしました」が正しく使えているか
- 日程確定後に復唱しているか

## 指導上の注意

> **文化背景の説明が不可欠：**
> タイ語の敬語は「語尾ひとつ」で完結するが、日本語は動詞・名詞・表現全体が変わる。
> この構造的な違いを最初に明示することで、学習者の混乱を減らせる。
>
> **英語訳を先に提示：**
> 日本語フレーズを提示する前に英語訳を見せることで、意味の理解が早まり、
> 次に日本語形式を覚えることに集中できる。
>
> **クッション言葉を重点指導：**
> タイ語では不要なクッション言葉が、日本語ビジネスでは相手への印象を大きく左右する。
> 「あるとないとで、どう聞こえるか」を実演して体感させる。
