## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　当社では商品の品質 <u>管理</u> に万全を期しております。
1. かんり　　2. かんけい　　3. かんごう　　4. がんり
> **正解：1（かんり）**　「管理」＝物事をきちんと取りしきること。「管」＝カン、「理」＝リ。「品質管理」「時間管理」のように使う。

**2**　万一 <u>不良</u> 品などがございましたら、ご連絡ください。
1. ふりょう　　2. ぶりょう　　3. ふろう　　4. ふりょ
> **正解：1（ふりょう）**　「不良」＝品質や状態がよくないこと。「不」＝フ、「良」＝リョウ。「不良品（ふりょうひん）」のように使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　このたびは商品を（　　）いただき、誠にありがとうございます。
1. お買い上げ　　2. お取り寄せ　　3. お申し込み　　4. お問い合わせ
> **正解：1（お買い上げ）**　「お買い上げ」＝客が商品を買うことを敬って言う表現。「お買い上げいただく」で購入のお礼に使う定型表現。「お問い合わせ」は質問すること。

**4**　商品の品質管理には（　　）を期しております。
1. 万全　　2. 完全　　3. 安全　　4. 健全
> **正解：1（万全）**　「万全を期す」＝少しも手抜かりがないように準備する慣用表現。「万全」は完全で手落ちがないこと。「安全」は危険がないこと、「健全」は健康で正常なこと。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　特別展を（　　）皆様は、入館前に入口の注意事項をお読みください。
1. ご覧になる　　2. ご覧に入れる　　3. 拝見する　　4. 拝見なさる
> **正解：1（ご覧になる）**　「ご覧になる」＝「見る」の尊敬語（相手・第三者の行為）。お客様が見る行為なので尊敬語を使う。「拝見する」は「見る」の謙譲語（自分の行為）。

**6**　先生から貴重な資料を（　　）ことは、私の研究の大きな助けになりました。
1. 拝見できました　　2. 拝借できました　　3. お貸しになりました　　4. お見えになりました
> **正解：2（拝借できました）**　「拝借する」＝「借りる」の謙譲語（自分が借りる）。先生から借りた（自分の行為）なので謙譲語。「お貸しになる」は貸す（相手）の尊敬語。

**7**　お忙しいところ恐縮ですが、貴社主催のツアーについて、詳細を教えて（　　）と存じます。
1. さしあげて　　2. さしあげれば　　3. いただいて　　4. いただければ
> **正解：4（いただければ）**　「〜ていただければ」＝〜してもらえれば（丁寧な依頼）。「教えていただければと存じます」は教えてほしいと丁寧にお願いする表現。「さしあげる」は自分が相手に与える場合。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　お客様が ＿＿ ＿＿ <span class="star">★</span> ＿＿ ください。
1. 2階の　　2. お見えになったら　　3. ご案内して　　4. 応接室に
> **正解：4（応接室に）**　正しい語順：「お見えになったら→2階の→★応接室に→ご案内して」。完成文：「お客様がお見えになったら、2階の応接室にご案内してください。」「お見えになる」＝「来る」の尊敬語。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　お買い上げ誠にありがとうございます。お届けいたしました商品の品質管理には万全を期しておりますが、万一不良品などがございましたら、お手数ですが、当社までご連絡くださいますようお願い申し上げます。

**9**　この文章は、何のために書かれたものか。
1. 商品購入のお礼と、不良品があった場合の連絡のお願い
2. 新商品の宣伝
3. 返品はできないという断り
4. 商品の値上げの案内
> **正解：1**　「お買い上げ誠にありがとうございます」とお礼を述べ、「万一不良品などがございましたら…ご連絡くださいますようお願い申し上げます」と不良品時の連絡を依頼している。

**10**　不良品があった場合、客はどうすればよいか。
1. そのまま使う。
2. 当社（販売した会社）に連絡する。
3. 自分で店に返品に行く。
4. 何もしなくてよい。
> **正解：2**　「当社までご連絡くださいますようお願い申し上げます」とあり、不良品の際は会社へ連絡するよう求めている。
