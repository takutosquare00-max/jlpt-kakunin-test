# N2文法 ランダムテスト No.08（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　この防災セットに入っているご飯は、常温で5年間保存できる（　　）。
1. かのようです　　2. ようにしています　　3. ようになっています　　4. ようにしてください

**問2**　来週の焼き肉パーティーは参加費500円で（　　）食べられますから、皆さん参加してください。
1. 好きだから　　2. 好きなのに　　3. 好きなだけ　　4. 好きなくらい

**問3**　絶滅する（　　）動植物の保護に関する条約が新しく作られたそうだ。
1. かわりに　　2. はずがない　　3. ことがある　　4. おそれがある

**問4**　商品が ＿＿ ＿＿ ★ ＿＿ と思います。
1. 入荷するまで　　2. いただければ　　3. お待ち　　4. しばらく

**問5**　料理研究家の栗林さんは独創的な創作料理を発表する（　　）、各地の伝統的な郷土料理の研究もされています。
1. 一方　　2. 次第　　3. 際　　4. 限り

**問6**　全力を尽くしてがんばれば、たとえ（　　）後悔はしないだろう。
1. 失敗しなければ　　2. 失敗したくないので　　3. 失敗しても　　4. 失敗したのに

**問7**　新しい生活を始める（　　）、大学の近くに部屋を探すことにした。
1. において　　2. にしたがって　　3. に基づいて　　4. にあたって

**問8**　警備を強化した（　　）、３億円のダイヤが盗まれ、警備が適切だったか問題になっている。
1. ほうが　　2. からには　　3. ことなく　　4. にもかかわらず

**問9**　彼は ＿＿ ＿＿ ★ ＿＿ と言われている。
1. 笑わせること　　2. にかけては　　3. 人を　　4. 天才だ

**問10**　宇宙から地球を見ると、国境など地図上に ＿＿ ＿＿ ★ ＿＿ と実感する。
1. にすぎない　　2. 書かれた　　3. 単なる　　4. 線

---

## 解答と解説

**問1**　正解：**3**（ようになっています）
「〜ようになっている」＝そういう仕組み・設計になっている。「5年間保存できるようになっている」はそのように製造されているという意味。「ようにしている」は自分が意識的にそうしているという意味。

**問2**　正解：**3**（好きなだけ）
「〜だけ」＝望む量・程度で（制限なく）。「好きなだけ食べられる」は食べたい分だけ食べてよいという意味。「〜なのに」は逆接、「〜くらい」は程度の比較を表す。

**問3**　正解：**4**（おそれがある）
「〜おそれがある」＝〜の危険・心配がある。「絶滅するおそれがある動植物」は絶滅しそうな危険がある動植物という意味。「ことがある」は〜することがある（可能性）という意味で危険の意味はない。

**問4**　正解：**3**（お待ち）
正しい語順：「入荷するまで→しばらく→★お待ち→いただければ」。完成文：「商品が入荷するまで、しばらくお待ちいただければと思います。」「〜ていただければ」＝〜してもらえれば（丁寧な依頼）。

**問5**　正解：**1**（一方）
「〜一方（で）」＝〜しながら別のことも・〜すると同時に。「発表する一方、研究もしている」は二つのことを同時に行っているという意味。「次第」は〜したらすぐに（完了後の即座の行動）。

**問6**　正解：**3**（失敗しても）
「たとえ〜ても」＝〜であっても（逆接の仮定）。「たとえ失敗しても後悔はしない」はもし失敗したとしても後悔しないという意味。「たとえ」は仮定の逆接と一緒に使うことが多い。

**問7**　正解：**4**（にあたって）
「〜にあたって」＝〜するにあたり・〜する重要な場面で。「新しい生活を始めるにあたって」は新生活を始めるというタイミングに際して、という意味。「において」は場所・場面、「にしたがって」は変化の並行進行。

**問8**　正解：**4**（にもかかわらず）
「〜にもかかわらず」＝〜なのに・〜であるにもかかわらず（逆接・反対の結果）。「強化したにもかかわらず盗まれた」は予想外の結果を強調する。「ことなく」は〜せずに、を表す。

**問9**　正解：**2**（にかけては）
正しい語順：「人を→笑わせること→★にかけては→天才だ」。完成文：「彼は人を笑わせることにかけては天才だと言われている。」「〜にかけては」＝〜の点に関しては（得意分野の強調）。

**問10**　正解：**4**（線）
正しい語順：「書かれた→単なる→★線→にすぎない」。完成文：「国境など地図上に書かれた単なる線にすぎないと実感する。」「〜にすぎない」＝〜でしかない（過大評価を否定する）。
