# N2文法 ランダムテスト No.11（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　区民祭り（　　）お問い合わせは、下記事務局にお願いいたします。
1. からいうと　　2. に限る　　3. に関する　　4. にわたる

**問2**　＿＿ ＿＿ ★ ＿＿ 確認しておきましょう。
1. 際に　　2. 地震の　　3. 避難できる　　4. 場所を

**問3**　みどり市では市民の皆様（　　）環境保護に取り組んでいます。
1. にともない　　2. とともに　　3. にかけて　　4. 次第で

**問4**　このあたりは ＿＿ ＿＿ ★ ＿＿ いて、住みやすい。
1. 自然が　　2. にしては　　3. 都心　　4. 残って

**問5**　平和への願い（　　）参加者全員で歌を歌った。
1. を問わず　　2. をきっかけに　　3. を込めて　　4. をもとに

**問6**　今回のマラソンは大方の予想（　　）、初出場の無名のランナーが優勝した。
1. に関して　　2. に反して　　3. のみならず　　4. からいうと

**問7**　景気の ＿＿ ＿＿ ★ ＿＿ わが国の財政も苦しくなっています。
1. 悪化　　2. 減少し　　3. 税金収入が　　4. にともなって

**問8**　「いくら ＿＿ ＿＿ ★ ＿＿ 。」と母は言った。
1. 年は　　2. 若い子　　3. ごまかせなかった　　4. ぶっても

**問9**　このデータ（　　）男性のほうが女性より甘い飲み物を好む傾向があることがわかります。
1. から見ると　　2. に関して　　3. にしたがって　　4. のみならず

**問10**　今回のサミット ＿＿ ＿＿ ★ ＿＿ と首相は語った。
1. における　　2. である　　3. エネルギー問題　　4. 最重要課題は

---

## 解答と解説

**問1**　正解：**3**（に関する）
「〜に関する」＝〜についての（名詞修飾形）。「区民祭りに関するお問い合わせ」は区民祭りについての問い合わせという意味。「に関して」の連体修飾形として名詞の前で使う。

**問2**　正解：**3**（避難できる）
正しい語順：「地震の→際に→★避難できる→場所を」。完成文：「地震の際に避難できる場所を確認しておきましょう。」「〜の際に」＝〜のとき（改まった表現）。

**問3**　正解：**2**（とともに）
「〜とともに」＝〜と一緒に。「市民の皆様とともに」は市民と力を合わせてという意味。「にともなって」は〜の変化に伴いという意味で使い方が異なる。

**問4**　正解：**1**（自然が）
正しい語順：「都心→にしては→★自然が→残って」。完成文：「このあたりは都心にしては自然が残っていて、住みやすい。」「〜にしては」＝〜の割には（期待よりいい意味でも使う）。

**問5**　正解：**3**（を込めて）
「〜を込めて」＝気持ち・感情を入れて。「願いを込めて歌を歌った」は平和への願いを気持ちに乗せて歌ったという意味。「をもとに」は〜を基にして（根拠・材料）。

**問6**　正解：**2**（に反して）
「〜に反して」＝〜に反対の結果・〜と違って。「予想に反して」は予想とは逆の結果だったという意味。「に関して」は〜について（テーマ）、「からいうと」は〜の観点から判断すると。

**問7**　正解：**3**（税金収入が）
正しい語順：「悪化→にともなって→★税金収入が→減少し」。完成文：「景気の悪化にともなって税金収入が減少し、わが国の財政も苦しくなっています。」「〜にともなって」＝〜の変化に合わせて。

**問8**　正解：**1**（年は）
正しい語順：「若い子→ぶっても→★年は→ごまかせなかった」。完成文：「いくら若い子ぶっても、年はごまかせなかった。」「いくら〜ても」＝どんなに〜しても（逆接の強調）。

**問9**　正解：**1**（から見ると）
「〜から見ると」＝〜を根拠・視点として見ると。「データから見ると」はデータを見る限りという意味。「に関して」は〜についてのテーマを表し、ここでは不自然。「にしたがって」は変化の並行進行。

**問10**　正解：**3**（エネルギー問題）
正しい語順：「における→最重要課題は→★エネルギー問題→である」。完成文：「今回のサミットにおける最重要課題はエネルギー問題であると首相は語った。」「〜における」＝〜での（場所・場面）。
