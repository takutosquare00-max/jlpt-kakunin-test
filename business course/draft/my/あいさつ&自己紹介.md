# ビジネスの挨拶まとめ（日本語学習者向け）

---

## 初対面の挨拶

**基本フレーズ**


| 場面         | フレーズ              |
| ---------- | ----------------- |
| 自己紹介       | はじめまして。〔名前〕と申します。 |
| 所属を伝える     | 〔会社名〕の〔名前〕と申します。  |
| よろしくお願いします | どうぞよろしくお願いいたします。  |


**名刺交換のとき**

- 渡すとき：「よろしくお願いいたします。」
- もらうとき：「頂戴いたします。」

---

## 2回目以降の挨拶

**基本フレーズ**


| 場面        | フレーズ            |
| --------- | --------------- |
| 再会したとき    | お世話になっております。    |
| 久しぶりに会うとき | ご無沙汰しております。     |
| 前回のお礼     | 先日はありがとうございました。 |


---

## 簡単な自己紹介（例文）

> タイから来た日本語学習者を想定したサンプルです。

**自己紹介の流れ**

1. 挨拶
2. 名前
3. 出身
4. 日本語について一言
5. よろしくお願いします

---

**例文（短め）**

> はじめまして。〔名前〕と申します。
> タイから参りました。
> 日本語を勉強中ですので、どうぞよろしくお願いいたします。

---

**例文（少し丁寧）**

> はじめまして。〔名前〕と申します。
> タイのバンコク出身です。
> 現在、日本語を勉強しております。
> まだ不慣れな点もございますが、どうぞよろしくお願いいたします。

---

**よく使う表現**


| 場面           | フレーズ             |
| ------------ | ---------------- |
| 出身を伝える       | タイから参りました。       |
| 勉強中であることを伝える | 日本語を勉強中です。       |
| 謙遜する         | まだ不慣れな点もございます。   |
| 締めの言葉        | どうぞよろしくお願いいたします。 |


---

## 会話シーン

### シーン①：初対面（ビジネスイベントにて）

> 登場人物：田中さん（日本人）、ソムチャイさん（タイ人・日本語学習中）

---

**田中**：はじめまして。田中と申します。よろしくお願いいたします。

**ソムチャイ**：はじめまして。ソムチャイと申します。タイから参りました。日本語を勉強中ですので、よろしくお願いいたします。

**田中**：そうですか！日本語、お上手ですね。

**ソムチャイ**：ありがとうございます。まだまだです。

**田中**：いえいえ。何かあれば、気軽に声をかけてください。

**ソムチャイ**：ありがとうございます。よろしくお願いします。

---

### シーン②：2回目の出会い（社内・廊下にて）

> 前回会ったことがある場面

---

**田中**：あ、ソムチャイさん、お世話になっております。

**ソムチャイ**：田中さん、お世話になっております。先日はありがとうございました。

**田中**：こちらこそ。最近、日本語の調子はどうですか？

**ソムチャイ**：少しずつ、上手になっていると思います（笑）

**田中**：それは良かった。また何かあれば、いつでもどうぞ。

**ソムチャイ**：はい、ありがとうございます。よろしくお願いします。

---

### シーン③：社内ですれ違うとき

> 同じ会社の人に廊下や休憩室で会った場面

---

**田中**：ソムチャイさん、お疲れ様です。

**ソムチャイ**：田中さん、お疲れ様です。

---

### シーン④：帰るとき・仕事が終わったとき

---

**田中**：お先に失礼します。

**ソムチャイ**：お疲れ様でした。お気をつけて。

---

**ソムチャイ**：田中さん、お疲れ様でした。お先に失礼します。

**田中**：お疲れ様でした。またね。

---

### お疲れ様の使い分け


| フレーズ    | 使うタイミング           |
| ------- | ----------------- |
| お疲れ様です  | 仕事中に同僚・上司と会ったとき   |
| お疲れ様でした | 仕事が終わったとき・相手が帰るとき |


> ⚠️ 「ご苦労様」は上司が部下に使う言葉なので、目上の人には使わないこと。

---

### よく使う返し言葉


| 相手の言葉         | 返し方                   |
| ------------- | --------------------- |
| 「日本語、お上手ですね。」 | 「ありがとうございます。まだまだです。」  |
| 「どうぞよろしく。」    | 「こちらこそ、よろしくお願いいたします。」 |
| 「先日はありがとう。」   | 「こちらこそ、ありがとうございました。」  |
| 「お疲れ様です。」     | 「お疲れ様です。」             |
| 「お先に失礼します。」   | 「お疲れ様でした。お気をつけて。」     |


---

## ポイント

- **「お世話になっております」** は2回目以降のビジネスの挨拶で最もよく使う万能フレーズ
- お辞儀は **15〜30度** が基本（深いほど丁寧）
- 相手より先に挨拶するのがマナー
