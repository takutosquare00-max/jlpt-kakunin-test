91うちの犬いぬは、雨が＿＿＿（A）いても、散歩さんぽに＿＿＿（B）がります。
★ 解答：（A）降って　（B）行きた
◆ 解説：「雨が降っていても」（逆接）＋「散歩に行きたがる」（犬が望む行動）。「〜たがる」は第三者の欲求を表す表現。「降って」は「降る→降っていても」の流れから「降って」が自然。
92仕事を変えて、給料きゅうりょうが＿＿＿（A）反面はんめん、残業ざんぎょうが多く、家族かぞくと過ごす＿＿＿（B）が減ってしまった。
★ 解答：（A）増えた　（B）時間
◆ 解説：「〜反面」は「〜という良い面がある一方で」という対比の表現。給料が増えた（良い面）反面、時間が減った（悪い面）という構造。
93A：お腹なかが＿＿＿（A）たまらないよ。
　　B：そこのコンビニでパンでも＿＿＿（B）来て、食べたら？
★ 解答：（A）すいて　（B）買って
◆ 解説：（A）「お腹がすいてたまらない」＝お腹がとても空いている。（B）「買って来て食べたら？」＝コンビニで買ってきたらどうかという提案。
94A：この資料しりょうは＿＿＿（A）中に返したほうがいいですか。
　　B：その資料しりょうなら、明日でもさしつかえ＿＿＿（B）ですよ。
★ 解答：（A）今日の　（B）ない
◆ 解説：（A）「今日の中に返す」＝今日中に返す。（B）「さしつかえない」＝問題ない・差し支えない。「明日でもさしつかえない」＝明日でも構わない。
95孫まごから電話で、目指めざしていた大学だいがくの試験しけんに＿＿＿（A）したと聞き、私はうれしい＿＿＿（B）この上ない。
★ 解答：（A）合格　（B）こと
◆ 解説：（A）「合格した」＝試験に受かった。（B）「うれしいことこの上ない」＝これ以上ないほどうれしい。「〜この上ない」は最上級の感情を表す慣用表現。
96父ちちを空港くうこうへ迎むかえに行きたいので、明日、
【1. 会社　→　2. 休む　→　3. いただく】ませんか。
★ 解答：会社を休ませていただけ
◆ 解説：「会社を休ませていただけませんか」＝上司などに対して休むことを丁寧にお願いする表現。「〜させていただく」は謙譲語で許可を求める際に使う。
97日本に来てから1か月げつが経たったが、
私は【1. これといって　→　2. 困こまる　→　3. いる】ことはない。
★ 解答：これといって困っていること（は）ない
◆ 解説：「これといって」＝特にこれというものは。「これといって困っていることはない」＝特に困っていることはない、という意味。
98A：この仕事は簡単なので、早く終わりそうですね。
B：ええ。でも、【1. 簡単　→　2. 仕事　→　3. ほど】間違まちがえやすいから、気をつけましょう。
★ 解答：簡単な仕事ほど
◆ 解説：「〜ほど〜」＝〜であるほど（程度・比較）。「簡単な仕事ほど間違えやすい」＝簡単だからこそ油断して間違えやすいという逆説的な表現。
99私が歌手かしゅになりたいと言ったら、【1. 友人ゆうじん　→　2. 鼻はな　→　3. 笑うわらう】ました。
★ 解答：友人に鼻で笑われ
◆ 解説：「鼻で笑う」＝馬鹿にしてせせら笑う・軽蔑して笑う慣用句。「友人に鼻で笑われました」＝友人に馬鹿にされて笑われた。受身形（笑われ）にすることで主語（私）の立場が表れる。
100故郷こきょうの【1. あまりの　→　2. 変化へんか　→　3. 言葉ことば】出なかった。
★ 解答：あまりの変化に言葉が
◆ 解説：「あまりの〜に言葉が出ない」＝あまりにも〜すぎて、驚きや感動で言葉が出てこない、という慣用的な表現。「故郷のあまりの変化に言葉が出なかった」＝故郷がこんなにも変わってしまったことに驚いて言葉が出なかった。
