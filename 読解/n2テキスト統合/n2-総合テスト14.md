## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　マディソンは、目の見えないリリーをいつも <u>気遣</u>って歩いていた。
1. きづか　　2. きずか　　3. きつか　　4. きげん
> **正解：1（きづか）**　「気遣う」＝相手のことを心配して気を配る。訓読みで「きづか（う）」。「気を遣う」が一語になった言葉。「気遣い」の形でも使う。

**2**　家の事情で、2匹を犬の <u>保護</u> センターに預けることになった。
1. ほご　　2. ほうご　　3. ほこ　　4. ほうこ
> **正解：1（ほご）**　「保護」＝危険なものから守ること。「保」＝ホ、「護」＝ゴ。「保護センター」「動物保護」のように使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　家の事情で、2匹を犬の保護センターに（　　）なければならなくなった。
1. 預け　　2. 授け　　3. 任せ　　4. 届け
> **正解：1（預け）**　「預ける」＝物や人・動物を一時的に他人にゆだねて世話してもらう。「センターに預ける」のように使う。「授ける」は与える、「任せる」は仕事をゆだねる。

**4**　この話がテレビで（　　）されると、多くの視聴者からコメントが寄せられた。
1. 報道　　2. 放送　　3. 報告　　4. 通報
> **正解：1（報道）**　「報道」＝新聞やテレビで世の中の出来事を広く伝えること。「報道される」のように使う。「報告」は決まった相手に伝えること、「通報」は事件などを知らせること。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　プレッシャーで実力が出せなかったとき、自分に負けた気がして、（　　）。
1. くやしいようになっている　　2. くやしかったつもりだ　　3. くやしくてならなかった　　4. くやしいおそれがあった
> **正解：3（くやしくてならなかった）**　「〜てならない」＝〜で仕方ない（感情・感覚が強くて抑えられない）。「くやしくてならなかった」はくやしくてたまらなかったという意味。「〜てしかたない」「〜てたまらない」と同義。

**6**　これは苦労して探し回った（　　）、やっと見つけた本なんです。
1. あげく　　2. あまり　　3. 末に　　4. 限り
> **正解：3（末に）**　「〜末に」＝長い過程・苦労の後に（努力の結果）。「探し回った末にやっと見つけた」は苦労して最終的に見つけたという意味。「あげく」はよくない結果に使う。

**7**　彼の理想はわかるが、あまりに現実離れしているので、賛成し（　　）。
1. ぬいた　　2. がたい　　3. がちだ　　4. かねない
> **正解：2（がたい）**　「〜がたい」＝〜しにくい・〜できない（困難・不可能）。「賛成しがたい」は賛成することが難しいという意味。「かねない」はよくない可能性、「がちだ」は悪い傾向。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　彼は ＿＿ ＿＿ <span class="star">★</span> ＿＿ パーティーでは人気者だ。
1. 歌えば　　2. ダンスも　　3. できるので　　4. 歌も
> **正解：2（ダンスも）**　正しい語順：「歌も→歌えば→★ダンスも→できるので」。完成文：「彼は歌も歌えばダンスもできるので、パーティーでは人気者だ。」「〜も〜ば〜も」＝〜もできるし〜もできる（複数を並べる）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　リリーは、子犬のころ目の病気になり、目が見えなくなった。そんなリリーのためにマディソンは「盲導犬」を引き受けて、いつもリリーを気遣いつつ歩いていた。２匹は散歩も一緒なら寝るのも一緒だった。そんな２匹を温かく見守っていた飼い主だったが、家の事情で２匹を犬の保護センターに預けなければならなくなった。「できるものならこの２匹をずっと一緒にいさせてやりたい」と、センターのスタッフは新しい飼い主を探した。
>
> 　この話が報道されると、多くの視聴者から「たとえどんなことがあっても２匹を引き離さないで」といったコメントがいくつも寄せられた。その中には新たな飼い主として声をあげる人もいる。２匹は、まもなく一生を過ごせる家を見つけ、幸福な日を送るようになるだろう。

**9**　マディソンは、リリーのために何をしていたか。
1. 盲導犬の役を引き受け、いつもリリーを気遣って一緒に歩いていた。
2. リリーと食べ物を取り合って競争していた。
3. リリーをきらって近づかなかった。
4. リリーの代わりに飼い主に世話をされていた。
> **正解：1**　「マディソンは『盲導犬』を引き受けて、いつもリリーを気遣いつつ歩いていた」とある。目の見えないリリーを助けて一緒に行動していた。

**10**　この話が報道されて、どうなったか。
1. ２匹はすぐに引き離されてしまった。
2. 多くの人からコメントが寄せられ、新しい飼い主になりたいという人も現れた。
3. ２匹はそのまま保護センターに残ることになった。
4. 元の飼い主が戻ってきて引き取った。
> **正解：2**　「多くの視聴者から…コメントがいくつも寄せられた。その中には新たな飼い主として声をあげる人もいる」とある。反響があり新しい飼い主の候補も現れた。
