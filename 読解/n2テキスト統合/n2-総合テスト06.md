## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　引っ越しのために、たくさんの荷物を <u>整理</u> することになった。
1. せいり　　2. せいれい　　3. せいりつ　　4. しょうり
> **正解：1（せいり）**　「整理」＝乱れたものをきちんと片づけること。「整」＝セイ、「理」＝リ。「せいりつ（成立）」「しょうり（勝利）」と混同しないよう注意。

**2**　古い荷物の中から、思い出の <u>品</u> が次々に出てきた。
1. しな　　2. ひん　　3. もの　　4. だな
> **正解：1（しな）**　「品」を単独で「品物」の意味で使うときは訓読みで「しな」。「思い出の品（しな）」のように読む。音読みの「ヒン」は「商品」「品質」など熟語で使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　祖父母も両親も物が捨てられない（　　）で、家には荷物が山のようにある。
1. 性格　　2. 性質　　3. 個性　　4. 特徴
> **正解：1（性格）**　「性格」＝その人の心の傾向・人がら。「捨てられない性格」のように人の傾向を表す。「性質」は物や事の本来の特徴、「個性」は他と違う独自の特徴。

**4**　結局、トラック１（　　）分の品を捨てることになった。
1. 杯　　2. 台　　3. 個　　4. 本
> **正解：1（杯）**　「トラック１杯分」＝トラックいっぱいになる量。容器や乗り物に満ちる量は「〜杯」で数える。「台」はトラックそのものを数える助数詞。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　彼は、自分のミスで仕事が遅れたのに、あれこれ言い訳した（　　）、結局一言も謝らなかった。
1. にしては　　2. からには　　3. ことなく　　4. あげく
> **正解：4（あげく）**　「〜あげく」＝さんざん〜した末に（よくない結果）。「言い訳したあげく謝らなかった」は、散々言い訳をして最終的にも謝らなかったという意味。「からには」は〜した以上は。

**6**　本場のタイ料理を作ってほしいと頼まれたが、材料がないので（　　）。
1. 作るおそれがある　　2. 作るわけではない　　3. 作りようがない　　4. 作るものだ
> **正解：3（作りようがない）**　「〜ようがない」＝どうしても〜できない（方法・手段がない）。「作りようがない」は材料がないので作ることが不可能という意味。

**7**　ABK社は一流企業（　　）給料が安くてびっくりした。
1. にしては　　2. はもとより　　3. を問わず　　4. さえ
> **正解：1（にしては）**　「〜にしては」＝〜の割には（期待外れ・予想と異なる）。「一流企業にしては給料が安い」は一流企業のわりに安かったという意外感を表す。「はもとより」は〜はもちろん。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　このあたりは ＿＿ ＿＿ <span class="star">★</span> ＿＿ いて、住みやすい。
1. 自然が　　2. にしては　　3. 都心　　4. 残って
> **正解：1（自然が）**　正しい語順：「都心→にしては→★自然が→残って」。完成文：「このあたりは都心にしては自然が残っていて、住みやすい。」「〜にしては」＝〜の割には（良い意味でも使う）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　引っ越しのために荷物を整理することになったが、祖父母も両親も物が捨てられない性格で、荷物が山のようにある。私たち姉妹の子どものときの物はもちろん、両親、祖父母の子ども時代の教科書まで出てきた。
>
> 　両親は古い荷物の中から思い出の品を手に取ってながめ、引っ越しどころではなく、さんざん昔話をしたあげく、父はすべて捨てないと言い出した。思い出の品とはいうものの、しまっておく場所もないので捨てざるを得ない。結局トラック１杯分の品を捨てた。父はさびしいかもしれないが、また新しい家で新しい思い出を作ってほしいと思う。

**9**　父が「すべて捨てない」と言い出したのは、なぜか。
1. 思い出の品をながめて昔話をしているうちに、捨てたくなくなったから
2. 新しい家がとても広いから
3. もともと荷物が少なかったから
4. 母に捨てるなと反対されたから
> **正解：1**　「思い出の品を手に取ってながめ…さんざん昔話をしたあげく、父はすべて捨てないと言い出した」とある。昔の品をながめ昔話をするうちに手放したくなくなったとわかる。

**10**　結局、荷物はどうなったか。
1. 全部そのまま残した。
2. トラック１杯分の品を捨てた。
3. 一つも捨てずに全部新居へ運んだ。
4. すべて祖父母の家に送った。
> **正解：2**　「しまっておく場所もないので捨てざるを得ない。結局トラック１杯分の品を捨てた」とある。捨てたくなかったが場所がなく、結局１杯分処分した。
