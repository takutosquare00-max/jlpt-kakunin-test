# N2文法 ランダムテスト No.04（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　私の町は気候が温暖な（　　）、海の幸も山の幸も豊富で暮らしやすい。
1. のに　　2. 上は　　3. 以上　　4. 上に

**問2**　今回のタンカーの事故で流出した油が幅５メートル、長さ300メートル（　　）広がり、環境汚染が心配されています。
1. にともなって　　2. について　　3. にわたって　　4. を中心に

**問3**　お忙しいところ恐縮ですが、詳細を教えて（　　）と存じます。
1. さしあげて　　2. さしあげれば　　3. いただいて　　4. いただければ

**問4**　仕事は長い時間働けばいい（　　）。効率よく進めることを考える必要がある。
1. に限る　　2. ということだ　　3. というものではない　　4. に決まっている

**問5**　注意書きを ＿＿ ＿＿ ★ ＿＿ 受け付けてもらえなかった。
1. 書類が　　2. ばかりに　　3. 見落としていた　　4. 足りなくて

**問6**　うちのチームは、何度もシュートを（　　）、得点することができなかった。
1. 試みながらも　　2. 試みて以来　　3. 試みたからといって　　4. 試みることなく

**問7**　奨学金を希望する場合はこの欄に丸印を付ける（　　）。
1. べき　　2. つもり　　3. こと　　4. はず

**問8**　梅を見に行ったが、名所と言われている（　　）、梅の木が少なくてがっかりした。
1. につけ　　2. ばかりでなく　　3. だけあって　　4. わりには

**問9**　企画会議で、部長の決定を本当にそれでいいのかと疑問に思い（　　）、聞いていた。
1. つつ　　2. ぬき　　3. がたくて　　4. を込めて

**問10**　明日は試験なのに、おなかが痛くて勉強（　　）。
1. するどころではない　　2. するわけではない　　3. することはない　　4. するものではない

---

## 解答と解説

**問1**　正解：**4**（上に）
「〜上に」＝〜だけでなく、それに加えて（好ましい追加）。「温暖な上に食材も豊富」は気候がいいうえに食材も恵まれているという意味。「のに」は逆接、「以上」は〜した以上は、を表す。

**問2**　正解：**3**（にわたって）
「〜にわたって」＝ある範囲・期間全体に。「幅5m、長さ300mにわたって広がった」は広い範囲に広がったという意味。「にともなって」は〜に伴い同時に変化する場合に使う。

**問3**　正解：**4**（いただければ）
「〜ていただければ」＝〜してもらえれば（丁寧な依頼）。「教えていただければと存じます」は教えてほしいと丁寧にお願いする表現。「さしあげる」は「あげる」の謙譲語（自分が相手に与える場合）。

**問4**　正解：**3**（というものではない）
「〜というものではない」＝〜とは限らない・必ずしも〜ではない。「長く働けばいいというものではない」は時間だけが重要ではないという意味。「に限る」は〜が一番だという意味。

**問5**　正解：**1**（書類が）
正しい語順：「見落としていた→ばかりに→★書類が→足りなくて」。完成文：「注意書きを見落としていたばかりに、書類が足りなくて受け付けてもらえなかった。」「〜ばかりに」＝〜だけが原因で（後悔）。

**問6**　正解：**1**（試みながらも）
「〜ながらも」＝〜しているのに・〜しているけれど（逆接）。「試みながらも得点できなかった」は、何度も試みたのにできなかったという意味。「試みることなく」は試みずに、という意味になってしまう。

**問7**　正解：**3**（こと）
「〜すること」＝指示・義務を表す書き言葉。案内文や規則でよく使われる。「べき」は強い義務、「つもり」は意志、「はず」は根拠ある推測を表す。

**問8**　正解：**4**（わりには）
「〜わりには」＝〜の割には（期待と実態のギャップ・がっかり）。「名所と言われているわりには梅の木が少ない」は評判ほどではなかったという失望を表す。「だけあって」は〜だけのことはある（評判通り）。

**問9**　正解：**1**（つつ）
「〜つつ」＝〜しながら（二つの状態・動作が並行）。「疑問に思いつつ聞いていた」は疑問を持ちながらも黙って聞いていたという意味。「〜ながら」とほぼ同意だがやや書き言葉的。

**問10**　正解：**1**（するどころではない）
「〜どころではない」＝〜できる状況・余裕がない。「勉強するどころではない」は痛くて勉強どころの話じゃないという意味。「することはない」は〜する必要はない（義務の否定）。
