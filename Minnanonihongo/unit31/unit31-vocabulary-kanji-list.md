# 第31課　漢字・語彙（クイズ用・K＝40）

`generate_unit31_sources.py` の `build_kanji` と **語順・語数** を一致させます。  
出所は `unit31.md` の **単語**（表・脚注※のうち **バリ・ピカソ・のぞみは出題しない**）／**《会話》**／**《読み物》**／**専門語彙**（表を左→右・上→下）です。副詞 **ずっと** は出題しません。

**出題しない語（K=40 のため表から外したもの）**: 式・帰り・お子さん・号・方・月・村・子ども・新神戸。  
**出題しない語（方針）**: 専門語彙の **コンピューター工学**（旧72語リスト #67）— 以降の語もクイズ対象外。
