# N2文法 ランダムテスト No.02（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　B：＿＿ ＿＿ ★ ＿＿ ことが大切です。がんばってね。
1. やってみる　　2. かどうか　　3. できる　　4. はともかく

**問2**　あきらめる（　　）、努力を続ければ、必ず成果は表れると信じている。
1. ことなく　　2. からには　　3. ものだから　　4. とおり

**問3**　退職の（　　）に必要な手続きについて、ご説明します。
1. たび　　2. 最中　　3. ところ　　4. 際

**問4**　彼女は今 ＿＿ ＿＿ ★ ＿＿ テレビにも出ている。
1. よく　　2. バイオリニスト　　3. 注目の　　4. だけあって

**問5**　B：ほんと、死んだ（　　）と思うくらいきつかったよ。
1. ほうがましだ　　2. ところだった　　3. ことだ　　4. おそれがある

**問6**　立場（　　）、寮長の私が当番をサボるわけにはいかないんです。
1. において　　2. に反して　　3. 上　　4. に関して

**問7**　健康の ＿＿ ＿＿ ★ ＿＿ でも、栄養のバランスが取れていないことがある。
1. 気をつけている　　2. ため　　3. つもり　　4. 食べ物に

**問8**　当店では、メーカー ＿＿ ＿＿ ★ ＿＿ いたします。
1. どんな　　2. 下取り　　3. パソコンでも　　4. を問わず

**問9**　ただ今、こちらの商品（　　）、全国どこでも210円で配送を承ります。
1. を問わず　　2. の際は　　3. に限り　　4. に応じて

**問10**　どんな仕事でも自分でやってみない（　　）その大変さはわからないだろう。
1. として　　2. 限り　　3. 際　　4. というより

---

## 解答と解説

**問1**　正解：**4**（はともかく）
正しい語順：「できる→かどうか→★はともかく→やってみる」。完成文：「できるかどうかはともかく、やってみることが大切です。」「〜はともかく」＝〜はさておき（一旦置いておいて）。

**問2**　正解：**1**（ことなく）
「〜ことなく」＝〜せずに・〜しないで。「あきらめることなく努力を続ければ」はあきらめずに続ければという意味。「からには」は〜した以上は、「とおり」は〜の通りに、を表す。

**問3**　正解：**4**（際）
「際（に）」＝〜のときに（改まった書き言葉）。「退職の際に」は退職するときにという意味。「たびに」は毎回、「最中」は〜の最中、「ところ」は〜しようとしているときに使う。

**問4**　正解：**4**（だけあって）
正しい語順：「注目の→バイオリニスト→★だけあって→よく」。完成文：「彼女は今注目のバイオリニストだけあって、よくテレビにも出ている。」「〜だけあって」＝〜だから当然（評判通り）。

**問5**　正解：**1**（ほうがましだ）
「〜ほうがましだ」＝〜の方がまだいい（不快な状況の強調）。「死んだほうがましだと思うくらいきつかった」は非常につらかったという気持ちを誇張して表現している。

**問6**　正解：**3**（上）
「〜上（じょう）」＝〜の観点から・〜の理由から・〜を考えると。「立場上」は自分の立場を考えるとという意味。「において」は場所・場面、「に反して」は〜に逆らって、を表す。

**問7**　正解：**1**（気をつけている）
正しい語順：「ため→食べ物に→★気をつけている→つもり」。完成文：「健康のため食べ物に気をつけているつもりでも、栄養のバランスが取れていないことがある。」「〜つもりでも」＝〜と思っていても（実態は違う）。

**問8**　正解：**3**（パソコンでも）
正しい語順：「を問わず→どんな→★パソコンでも→下取り」。完成文：「メーカーを問わずどんなパソコンでも下取りいたします。」「〜を問わず」＝〜に関係なく。

**問9**　正解：**3**（に限り）
「に限り」＝〜だけ（限定）。「この商品に限り」はこの商品だけという意味。「に応じて」は程度に合わせる、「の際は」はそのときは、という意味。

**問10**　正解：**2**（限り）
「〜ない限り」＝〜しない間は・〜しなければ（条件）。「やってみない限りわからない」はやってみなければわからないという意味。「〜限り」は肯定形と否定形で意味が異なる点に注意。
