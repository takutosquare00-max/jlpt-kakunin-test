# N2文法 ランダムテスト No.14（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　健康面のみならず、仕事の能率という点からも、 ＿＿ ＿＿ ★ ＿＿ はいいことだと思う。
1. 生活習慣が　　2. の　　3. 見直されつつある　　4. 早起きの

**問2**　野菜や果物に ＿＿ ＿＿ ★ ＿＿ 減っているのをご存知ですか。
1. ビタミンが　　2. 昔　　3. 含まれる　　4. に比べて

**問3**　ぼくが花瓶を割ったとき、お母さんにしかられないように、＿＿ ＿＿ ★ ＿＿ くれた。
1. 割った　　2. 猫が　　3. お兄ちゃんが　　4. ことにして

**問4**　商品の価格は、売れ残りなどのリスクを考えた（　　）設定されるものである。
1. としたら　　2. 上で　　3. にすぎず　　4. わりに

**問5**　バイオリンは弾く（　　）歌うような感覚が大事です。
1. として　　2. といえば　　3. というより　　4. というと

**問6**　暑いから、狭いテントで ＿＿ ＿＿ ★ ＿＿ ほうがましだ。
1. くらいなら　　2. 寝た　　3. 外で　　4. 寝る

**問7**　工場建設（　　）、部長は打ち合わせのためにロシアへ向かった。
1. において　　2. に先立って　　3. にわたって　　4. につけ

**問8**　当店では、作り（　　）の味をお楽しみいただけるよう、ご注文を受けてから作っています。
1. ながら　　2. たて　　3. かけ　　4. しだい

**問9**　数年前に命にかかわるような ＿＿ ＿＿ ★ ＿＿ 注意するようになった。
1. 以来　　2. 大病を　　3. 健康に　　4. して

**問10**　「独立すると言って会社を辞める（　　）、どんなに大変でもがんばれ」と部長に言われた。
1. ことなく　　2. からには　　3. ように　　4. おかげで

---

## 解答と解説

**問1**　正解：**3**（見直されつつある）
正しい語順：「早起きの→生活習慣が→★見直されつつある→の」。完成文：「早起きの生活習慣が見直されつつあるのはいいことだと思う。」「〜つつある」＝変化が進行中。

**問2**　正解：**2**（昔）
正しい語順：「含まれる→ビタミンが→★昔→に比べて」。完成文：「野菜や果物に含まれるビタミンが昔に比べて減っているのをご存知ですか。」「〜に比べて」＝〜と比較して。

**問3**　正解：**1**（割った）
正しい語順：「お兄ちゃんが→猫が→★割った→ことにして」。完成文：「お兄ちゃんが猫が割ったことにしてくれた。」「〜ことにする」＝実際とは違うことを〜と決める・見なす（事実の隠蔽・フィクション）。

**問4**　正解：**2**（上で）
「〜た上で」＝〜してから・〜した後で（段階・手順）。「リスクを考えた上で価格が設定される」はリスクを検討してから価格を決めるという意味。「としたら」は仮定、「わりに」は割には（ギャップ）。

**問5**　正解：**3**（というより）
「〜というより」＝〜というよりも（より正確な言い換え）。「弾くというより歌うような感覚」は、弾くというイメージよりも歌うほうが近いという意味。

**問6**　正解：**3**（外で）
正しい語順：「寝る→くらいなら→★外で→寝た」。完成文：「狭いテントで寝るくらいなら外で寝たほうがましだ。」「〜くらいなら〜ほうがましだ」＝〜するくらいだったら別の方がまだいい。

**問7**　正解：**2**（に先立って）
「〜に先立って」＝〜に先行して・〜の前に。「工場建設に先立って打ち合わせに行った」は建設開始の前に準備のため行ったという意味。「において」は場所・場面、「にわたって」は範囲の広がり。

**問8**　正解：**2**（たて）
「〜たて」＝〜したばかり（新鮮・新しい状態）。「作りたての味」は作ったばかりで新鮮な味という意味。「〜かけ」は〜しかけた・途中の状態、「〜ながら」は〜しながら（同時進行）。

**問9**　正解：**1**（以来）
正しい語順：「大病を→して→★以来→健康に」。完成文：「大病をして以来、健康に注意するようになった。」「〜して以来」＝〜してからずっと（継続）。

**問10**　正解：**2**（からには）
「〜からには」＝〜と決めた以上は・〜した以上は（決意・覚悟）。「辞めるからには」は辞めると決めた以上はという意味。「ことなく」は〜せずに、という意味になる。
