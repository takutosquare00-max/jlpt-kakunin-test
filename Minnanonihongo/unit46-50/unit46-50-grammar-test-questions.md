# 解答一覧

## パート1 解答

| 問 | 正解 | 文法ポイント | 課 |
|---|---|---|---|
| 問1 | **a. ところ** | 動詞辞書形＋ところ：直前（これから〜する寸前） | 46 |
| 問2 | **b. ばかり** | 動詞た形＋ばかり：時間的近接（〜したばかり） | 46 |
| 問3 | **c. はず** | 普通形＋はず：確信に基づく推量（〜のはずだ） | 46 |
| 問4 | **b. そう**（c も別解） | 普通形＋そうだ：伝聞。〜らしい も推量・伝聞で可。 | 47 |
| 問5 | **c. よう／b. はず** | 〜ようだ：観察に基づく推量／〜はずだ：根拠ある確信。元気がない→何かあった よう／はず どちらも自然 | 46・47 |
| 問6 | **b. が** | 〜が します：感覚（匂い・音・味・声・気が する） | 47 |
| 問7 | **b. せ** | 使役形：習う→習わせる（u動詞 う段→あ段＋せる） | 48 |
| 問8 | **a. に** | 〜に＋他動詞使役：対象に〜させる（学生に作文を書かせる） | 48 |
| 問9 | **a. せていただけません** | 〜させていただけませんか：謙譲使役の依頼（休ませていただけませんか） | 48 |
| 問10 | **b. なっています** | お／ご〜になります：尊敬語の構文（お出かけになる） | 49 |
| 問11 | **b. 召し上がります** | 特別な尊敬語：食べる／飲む→召し上がる | 49 |
| 問12 | **b. ください** | お／ご〜ください：丁寧な指示（お座りください） | 49 |
| 問13 | **a. お描きになった** | お／ご〜になる：尊敬構文（先生が描く→お描きになる） | 49 |
| 問14 | **b. 申します** | 特別な謙譲語：言う→申す（自己紹介の定型） | 50 |
| 問15 | **a. お持ちします** | お／ご〜します：謙譲構文（自分が持つ→お持ちする） | 50 |
| 問16 | **b. 拝見します** | 特別な謙譲語：見る→拝見する | 50 |
| 問17 | **a. ところ** | 動詞て形＋いる＋ところ：最中（〜している最中） | 46 |
| 問18 | **b. 尊敬：いらっしゃいます／謙譲：参ります** | 行く・来る・いるの特別敬語：尊敬＝いらっしゃる、謙譲＝参る | 49,50 |

## 文法まとめ（Unit46〜50）

| 課 | 文法項目 | 形 | 例 |
|---|---|---|---|
| 46 | 〜ところです | 動詞辞書形＋ところ（直前）／動詞ている＋ところ（最中）／動詞た形＋ところ（直後） | 出かけるところ／食べているところ／着いたところ |
| 46 | 〜たばかりです | 動詞た形＋ばかり | 来たばかり・買ったばかり |
| 46 | 〜はずです | 普通形＋はず（い形：い／な形：な／名詞：の） | 来るはず・上手なはず・学生のはず |
| 47 | 〜そうです（伝聞） | 普通形＋そうだ | 雨が降るそう・忙しいそう |
| 47 | 〜ようです | 普通形＋ようだ（な形：な／名詞：の） | 風邪のようだ・元気なよう・着いたよう |
| 47 | 〜が します | 名詞（匂い／音／味／声／気／気持ち）＋が＋する | いい匂いがする・音がする |
| 48 | 使役形（u動詞） | う段→あ段＋せる | 書く→書かせる、読む→読ませる |
| 48 | 使役形（ru動詞） | る→させる | 食べる→食べさせる |
| 48 | 使役形（不規則） | する→させる、来る→来させる | 勉強させる・来させる |
| 48 | 使役の助詞 | 自動詞：〜を／他動詞：〜に＋〜を | 子どもを遊ばせる／学生に本を読ませる |
| 48 | 〜させていただけませんか | 動詞使役て形＋いただけませんか | 帰らせていただけませんか |
| 49 | 尊敬の活用 | u動詞：う段→あ段＋れる／ru動詞：られる／irr：される・こられる | 書かれる・食べられる・来られる |
| 49 | 特別な尊敬語 | いらっしゃる・召し上がる・ご覧になる・なさる・おっしゃる・くださる | 召し上がってください |
| 49 | お／ご〜ください | お＋動詞ます形＋ください／ご＋漢語＋ください | お入りください・ご利用ください |
| 49 | お／ご〜になります | お＋動詞ます形＋になる／ご＋漢語＋になる | お読みになる・ご出発になる |
| 50 | お／ご〜します／いたします | お＋動詞ます形＋する／ご＋漢語＋する（さらに丁寧：いたす） | お持ちします・ご案内いたします |
| 50 | 特別な謙譲語 | 参る・うかがう・申す・いたす・拝見する・存じる・おる・いただく | わたしは田中と申します |
| 50 | 尊敬 vs 謙譲 | 相手の動作→尊敬／自分の動作→謙譲 | 召し上がる（相手）／いただく（自分） |
