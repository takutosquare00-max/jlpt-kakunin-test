もんだい1 — 漢字の読み（2問）

＿＿の言葉の読み方として最もよいものを、1・2・3・4から一つ選んでください。

1駅伝部は１日も休むことなく練習に 励んできた。
✅ 正解：1（はげ）　「励む」＝一生懸命努力する。「励」の訓読みは「はげ（む）」。「挑（いど）む」「努（つと）める」「臨（のぞ）む」と混同しないよう注意。
2今年、ついに箱根駅伝出場の夢を 実現 することができた。
✅ 正解：3（じつげん）　「実現」＝願いや計画が現実になること。「実」＝ジツ、「現」＝ゲン。「じっけん（実験）」と読み間違えやすい。
もんだい2 — 語彙（2問）

（　）に入れるのに最もよいものを、1・2・3・4から一つ選んでください。

3他大学の選手と（　　）しても、わが校の練習量は決して負けていない。
✅ 正解：1（比較）　「比較」＝二つ以上を比べること。「他大学と比較する」は他校と比べるという意味。「比例」は一方が増えると他方も増える関係、「対象」は働きかける相手。
4これまで応援してくれた人への（　　）の気持ちを忘れず、全力で走りたい。
✅ 正解：2（感謝）　「感謝」＝ありがたく思う気持ち。「感謝の気持ち」でお礼の心を表す。「感想」は思ったこと、「感動」は心を動かされること。
もんだい3 — 文法（3問）
5うちのチームは、何度もシュートを（　　）、得点することができなかった。
✅ 正解：1（試みながらも）　「〜ながらも」＝〜しているのに・〜しているけれど（逆接）。「試みながらも得点できなかった」は、何度も試みたのにできなかったという意味。「試みることなく」は試みずに、という意味になる。
6ABKハウジングは東京（　　）、関東地方６県の不動産を取り扱っております。
✅ 正解：3（をはじめ）　「〜をはじめ」＝〜を代表として（他にも）。「東京をはじめ」は東京を筆頭に関東各県も、という意味。「のもとで」は〜の下で、「を目的に」は目的を表す。
7「独立すると言って会社を辞める（　　）、どんなに大変でもがんばれ」と部長に言われた。
✅ 正解：2（からには）　「〜からには」＝〜と決めた以上は・〜した以上は（決意・覚悟）。「辞めるからには」は辞めると決めた以上はという意味。「ことなく」は〜せずに、という意味になる。
もんだい4 — 文の組み立て（1問）
8数年前に命にかかわるような ＿＿ ＿＿ ★ ＿＿ 注意するようになった。
✅ 正解：1（以来）　正しい語順：「大病を→して→★以来→健康に」。完成文：「大病をして以来、健康に注意するようになった。」「〜して以来」＝〜してからずっと（継続）。
もんだい5 — 読解（2問・1文章）

皆様、本日は私たちABK大学駅伝部のためにお集まりいただき、ありがとうございます。

わが駅伝部は箱根駅伝※出場を目指して、１日も休むことなく練習に励んできました。他大学の選手と比べて、努力が足りなかったわけではないと思いますが、残念ながらこれまでわが校は箱根駅伝の出場権を得ることはありませんでした。でも、今年、私にとっては大学生活最後の年に、ついに夢を実現することができました。今回の出場をきっかけとして、駅伝部は大きく成長できると期待しています。

コーチをはじめ、これまで応援してくださった方のためにも、選手一同、精一杯がんばるつもりです。皆様、応援どうぞよろしくお願いいたします。

※箱根駅伝：東京〜箱根間を往復する大学対抗駅伝競技。

9ABK大学駅伝部について、正しいものはどれか。
✅ 正解：2　「これまでわが校は箱根駅伝の出場権を得ることはありませんでした」「ついに夢を実現することができました」とあり、今回初めて出場が決まったとわかる。出場はこれからなので3は誤り。
10この会で、誰が誰に話しているか。
✅ 正解：4　「私にとっては大学生活最後の年」から話し手は選手（最終学年）。「お集まりいただき」「皆様、応援どうぞよろしく」から、集まった応援者（参加者）に向けて話している。
