## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　新米ドライバーの私にとって、カーナビは <u>必需品</u> だ。
1. ひつじゅひん　　2. ひつようひん　　3. ひつじゅうひん　　4. ひっじゅひん
> **正解：1（ひつじゅひん）**　「必需品」＝なくてはならない品物。「必」＝ヒツ、「需」＝ジュ、「品」＝ヒン。「必要品」と書き間違えないよう注意。

**2**　カーナビは目的地まで道 <u>案内</u> をしてくれる。
1. あんない　　2. あんねい　　3. あんがい　　4. かんない
> **正解：1（あんない）**　「案内」＝道や場所を教え導くこと。「案」＝アン、「内」＝ナイ。「道案内」「館内案内」のように使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　目的地の情報を（　　）すれば、地図と音声で道案内をしてくれる。
1. 入力　　2. 入場　　3. 入手　　4. 入会
> **正解：1（入力）**　「入力」＝機器に文字や数値を打ち込むこと。「情報を入力する」のように使う。「入手」は手に入れること、「入場」は場所に入ること、「入会」は会に加わること。

**4**　ちょっと道を（　　）ただけで、彼女はぶつぶつ文句を言う。
1. 間違え　　2. 思い違え　　3. 勘違い　　4. 食い違え
> **正解：1（間違え）**　「道を間違える」＝正しくない道へ進んでしまう。「間違える」は誤ること。「勘違い」は思い込みによる誤り（「道を勘違える」とは言わない）。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　ブレーキの音が聞こえた（　　）、突然トラックが店に飛び込んできた。
1. かと思ったら　　2. かのように　　3. といっても　　4. からといって
> **正解：1（かと思ったら）**　「〜かと思ったら」＝〜したと思った途端に（すぐ次のことが起きる）。「音が聞こえたかと思ったらトラックが来た」は聞こえた瞬間に入ってきたという意味。

**6**　入学以来成績がトップだった彼は、今回のテストで2位になって、人生が終わってしまった（　　）落ち込んでいる。
1. かと思うと　　2. かのように　　3. 限り　　4. わりに
> **正解：2（かのように）**　「〜かのように」＝まるで〜のように（実際とは違う比喩）。「人生が終わったかのように落ち込んでいる」は実際は終わっていないのに、まるでそのような状態だという意味。

**7**　梅を見に行ったが、名所と言われている（　　）、梅の木が少なくてがっかりした。
1. につけ　　2. ばかりでなく　　3. だけあって　　4. わりには
> **正解：4（わりには）**　「〜わりには」＝〜の割には（期待と実態のギャップ・がっかり）。「名所と言われているわりには梅の木が少ない」は評判ほどではなかったという失望を表す。「だけあって」は評判通り。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　彼は他人に ＿＿ ＿＿ <span class="star">★</span> ＿＿ ので、みんなに嫌がられている。
1. 甘い　　2. 厳しい　　3. 自分に　　4. わりに
> **正解：3（自分に）**　正しい語順：「厳しい→わりに→★自分に→甘い」。完成文：「彼は他人に厳しいわりに自分に甘いので、みんなに嫌がられている。」「〜わりに」＝〜の割には（ギャップ・矛盾）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　新米ドライバーの私にとって、カーナビは必需品だ。情報を入力すれば、地図と音声で目的地まで道案内をしてくれる。高速道路では料金を教えてくれるし、休憩を取った様子がないと「ちょっと休んだほうが…」と話しかけてくる。その声は本当に心配しているかのようだ。まさに有能な秘書だ。地図やらガイドブックやらたくさん抱えて車に乗り込み、ちょっと道を間違えただけで、ぶつぶつ言う彼女よりずっといい。だがその彼女もカーナビがあれば道に迷うことはあるまいと思っているらしく、地図を見ていたかと思ったら、いつの間にか寝ていることもある。そうしたら、秘書と2人きりのドライブだ。静かでいいと思う反面、そのドライブにはどこかさびしいものがある。

**9**　筆者は、カーナビをどのようなものだと感じているか。
1. 有能な秘書のようなもの
2. じゃまでうるさい機械
3. まったく役に立たない道具
4. こわくて使いたくない存在
> **正解：1**　「まさに有能な秘書だ」とあり、道案内や料金・休憩の案内をしてくれるカーナビを有能な秘書にたとえている。

**10**　一緒に乗っている「彼女」について、本文の内容に合うものはどれか。
1. 道に迷うことは絶対にないと自信を持っている。
2. カーナビがあれば迷わないと思い、地図を見ているうちに寝てしまうことがある。
3. カーナビが嫌いで使わせない。
4. いつも正確に道を案内してくれる。
> **正解：2**　「カーナビがあれば道に迷うことはあるまいと思っているらしく、地図を見ていたかと思ったら、いつの間にか寝ている」とある。安心して眠ってしまうことがある。
