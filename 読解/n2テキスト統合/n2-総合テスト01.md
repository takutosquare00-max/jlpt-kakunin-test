## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　工事の <u>騒音</u> が近隣の問題になっている。
1. ぞうおん　　2. そうおん　　3. さわおん　　4. そうね
> **正解：2（そうおん）**　「騒」＝ソウ、「音」＝オン。「騒音」＝うるさくて不快な音。「騒」を「ぞう」「さわ」と読まないよう注意。

**2**　新しい計画について慎重に <u>検討</u> する。
1. けんとう　　2. げんとう　　3. けんとお　　4. けんと
> **正解：1（けんとう）**　「検討」＝よく調べて考えること。「討」＝トウ。「げんとう」「けんと」は誤り。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　新しい技術の登場は、社会に大きな（　　）を与えた。
1. 観察　　2. 提案　　3. 影響　　4. 発見
> **正解：3（影響）**　「影響を与える」＝他のものに作用を及ぼす。「観察・提案・発見」は「与える」とは結びつかない。

**4**　工事の騒音について、近隣の住民から多くの（　　）が寄せられた。
1. 感想　　2. 苦情　　3. 希望　　4. 評価
> **正解：2（苦情）**　「苦情」＝不満や迷惑を訴える声。「寄せられる」と自然に結びつく。「感想・評価」は中立的で文脈に合わない。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　店を経営するなら、家賃や人件費など、売り上げの多い少ない（　　）毎月費用がかかることを考えなければならない。
1. に応じて　　2. に限り　　3. にかかわらず　　4. につき
> **正解：3（にかかわらず）**　「にかかわらず」＝〜に関係なく。「多い少ないにかかわらず」は売上の多少に関係なくという意味。

**6**　原子力発電所の安全性（　　）、世界各地で議論が行われている。
1. を込めて　　2. を問わず　　3. をめぐって　　4. をもとに
> **正解：3（をめぐって）**　「をめぐって」＝〜について（対立・議論の的として）。「安全性をめぐって議論」は安全性を中心に議論するという意味。

**7**　この製品の使用法（　　）は、ホームページをご覧ください。
1. について　　2. にあたって　　3. をめぐって　　4. に基づいて
> **正解：1（について）**　「について」＝〜に関して。「使用法について（詳しくは）」と情報の対象を示す。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　地域住民が新しいホテルの ＿＿ ＿＿ <span class="star">★</span> ＿＿ 起こしたそうだ。
1. 運動を　　2. 建設　　3. 反対　　4. をめぐって
> **正解：3（反対）**　正しい語順：「建設→をめぐって→★反対→運動を」。完成文：「地域住民が新しいホテルの建設をめぐって反対運動を起こしたそうだ。」「をめぐって」＝〜を対立点として。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　新たな高速道路建設をめぐって、現在様々な議論が行われている。新しい高速道路ができれば、地域経済が活発になると期待する人も多い。しかし、国民の幸福という基本理念に基づいて建設計画が作られたとしても、まずその費用をどうするのかが問題だ。資金問題に加えて、周辺地域への騒音や大気汚染をどうするかも検討しなければなるまい。建設にあたって、クリアしなければならない問題はまだ数多く、決定には時間がかかるものと思われる。

**9**　この文章で筆者が一番言いたいことは何か。
1. 高速道路を新しく作る必要があること
2. 高速道路ができれば、地域経済が活発になること
3. 高速道路ができれば、国民が幸福になること
4. 高速道路の建設には、さまざまな問題があること
> **正解：4**　筆者は「費用」「資金問題」「騒音や大気汚染」など、解決すべき問題を次々に挙げ、「クリアしなければならない問題はまだ数多く」と述べている。最も伝えたいのは建設に多くの課題があるという点。

**10**　筆者は高速道路建設の問題として、何を挙げているか。
1. 建設に反対する住民が多いこと
2. 費用や騒音・大気汚染などの問題があること
3. 地域経済がまったく活発にならないこと
4. 国民の幸福という理念が欠けていること
> **正解：2**　本文に「まずその費用をどうするのか」「資金問題に加えて、周辺地域への騒音や大気汚染をどうするか」とある。1の住民の反対や4の理念の欠如は本文に書かれていない。
