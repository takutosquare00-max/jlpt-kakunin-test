# JLPT N5 タイピング練習アプリ用 お題集

## アプリの目的
- タイピング練習ではなく日本語の定着を目的とする
- 繰り返し読む（インプット）ことで理解を深める
- 繰り返し入力する（アウトプット）ことで記憶に定着させる
- 日本語を脳に自然に定着させる

## 使用上の注意

1. **インプット重視**: まずお題を声に出して読む（音読）ことを推奨します
2. **アウトプット重視**: タイピングする際は意味を考えながら入力してください
3. **繰り返し練習**: 同じお題を何度も繰り返すことで自然に日本語が定着します
4. **文型の理解**: 各カテゴリの文型を理解してから練習すると効果的です
5. **段階的学習**: 簡単なものから始めて徐々に難しいものに挑戦してください
