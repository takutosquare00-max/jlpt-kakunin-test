# PDF文字起こし仕様書
## みんなの日本語 教科書ページ → Markdownファイル変換手順書

---

## 概要

本書は、「みんなの日本語」シリーズ等の日本語教材PDFをMarkdownファイルへ文字起こしする際の手順・ルールを定めたものです。

**PDFによって収録ページ・セクション構成は異なります。**  
本書のルールは「原則」であり、実際のPDF内容を優先してください。PDFに存在するセクション・情報はすべて書き起こし対象です。

---

## 1. 作業の基本フロー

1. PDFを開いて**全ページを通しで確認**する
2. どのセクションが何ページに収録されているかを把握する
3. 各セクションに付いている**トラック番号（丸囲み数字）をすべて控える**
4. PDFに存在するセクションを**上から順に・一つも省かず**書き起こす
5. 完了後に品質チェックを行う

---

## 2. トラック番号について（最重要）

### 2.1 トラック番号とは

PDFの各セクション冒頭に表示されている丸囲みの数字（例：㊿、196 など）は、音声CDのトラック番号です。

### 2.2 記載ルール

- **すべてのセクションのトラック番号を記載する**
- 文型・例文・会話も含め、トラック番号があれば必ず記載する
- 記法：`**数字**`（太字）を、セクション見出しの直下に単独行で記載する
- 練習A〜問題は番号を問題番号の前に連結する（例：`**200** 1.`）

### 2.3 よくある抜け漏れ

文型・例文・会話のトラック番号は忘れやすい。PDFを確認する際、これら冒頭3セクションのトラック番号を**最初に控えておく**こと。

```
## 文型

**◯◯◯**

（本文）
```

```
## 例文

**◯◯◯**

（本文）
```

```
## 会話

**◯◯◯**

### 会話タイトル

（本文）
```

### 2.4 ページ欠落時の対処

- PDFに存在しないページのトラック番号は、前後の番号から推定して記載する
- 推定したトラック番号は記載した上で、その下に注記を入れる

```
## 会話

**◯◯◯**

※ 会話ページはPDFに含まれていません。
```

## 3. ファイル構成

### 3.1 ファイル名・配置

各課は `Minnanonihongo/unitN/` フォルダにまとめ、本文は `unitN.md`。数字は半角。

### 3.2 全体構造

PDFに収録されているセクションをすべて含める。**収録されていないセクションは省略してよいが、収録されているセクションを省くことは禁止。**

標準的なセクション順：文型 → 例文 → 会話 → （単語） → 練習A → 練習B → 練習C → 問題

PDFによっては「単語」「文法説明」などが追加されることがある。それらも漏らさず書き起こす。

---

## 4. 各セクションの書き起こしルール

### 4.1 共通ルール

- ふりがな（ルビ）は**省略**し、漢字のみ記載する
- 助詞前後の全角スペース（　）はそのまま保持する
- 教材に存在する記号（`……`・`→`・`[へ]`・`（　）` 等）はそのまま保持する

### 4.2 文型・例文

- 番号付きリストで記述する
- 例文の応答文は `……` で始め、3スペースでインデントする

### 4.3 会話

- 発話者と台詞をMarkdownテーブルで記述する
- 場面の区切り（点線）は `……………………………………………………` で表現する
- 場面転換ごとにテーブルを分割する
- 発話者名はふりがななしで記載する

### 4.4 単語（収録されている場合）

- 表形式（Markdownテーブル）で記述する
- 品詞・意味（翻訳）など、PDFに記載されている列をすべて含める

### 4.5 練習A

- ピンクのハイライトボックスの内容はリスト（`-`）形式で縦並びに記述する
- 質問語（なに、どこ、だれ、いつ 等）は `**太字**` にする
- 活用表がある場合はMarkdownテーブルで記述する

### 4.6 練習B・C

- 例文（れい）を最初に記述し、練習問題を番号リストで記載する
- 練習Cの対話形式は `- A：` / `- B：` で記述する
- イラストの内容は `（絵：説明）` 形式で補足する

### 4.7 問題

- 記述式の解答欄は `___________________________________________` で表現する
- 選択問題は `1)（　）` 形式で記載する
- 問題文中の正解は `**太字**` で表現する
- 読解テキストは引用ブロック（`>`）で記載する

---

## 5. イラストの補足

文字起こし不可能な視覚要素は以下の形式で補足する：

```
（絵：内容の説明）
（地図：各地点の位置関係）
（カレンダー：きょう＝火曜日　木曜日＝会議）
```

練習問題を解くために必要な情報（時刻・場所・人物・行動など）を優先的に記載する。詳細な描写は不要。

---

## 6. 品質チェックリスト

書き起こし完了後、以下をすべて確認する：

- [ ] PDFの全ページを確認し、見落としたセクションがないか
- [ ] **文型・例文・会話のトラック番号が記載されているか**（抜けやすいので必ず確認）
- [ ] 練習A以降のトラック番号がすべて記載されているか
- [ ] トラック番号が連続しているか（欠番がある場合はページ欠落を疑う）
- [ ] 会話の場面区切り・テーブル分割が適切か
- [ ] 練習Aのハイライトボックスがリスト形式で表現されているか
- [ ] イラストの補足説明が含まれているか
- [ ] 解答欄（下線・括弧）が適切に表現されているか
- [ ] 読解テキストが引用ブロックで記載されているか
- [ ] 全角スペースが本文中に保持されているか
