## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　日本のサラリーマンには <u>睡眠</u> 不足の人が多い。
1. すいめん　　2. すいみん　　3. ずいみん　　4. すいねむ
> **正解：2（すいみん）**　「睡眠」＝眠ること。「睡」＝スイ、「眠」＝ミン。「すいめん（水面）」と混同しやすいので注意。

**2**　眠くなる時間帯には、短い <u>仮眠</u> を取るとよい。
1. かみん　　2. かりみん　　3. きゅうみん　　4. がみん
> **正解：1（かみん）**　「仮眠」＝短い時間だけ眠ること。「仮」＝カ（一時的）、「眠」＝ミン。「きゅうみん（休眠）」と意味・読みを区別する。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　午後２時を過ぎて眠くなると、仕事の（　　）が下がってしまう。
1. 能率　　2. 利益　　3. 成績　　4. 収入
> **正解：1（能率）**　「能率」＝一定時間にできる仕事の量・効率。「仕事の能率が下がる」ははかどらなくなるという意味。「利益・収入」はお金、「成績」は学業や評価。

**4**　眠気を（　　）するには、深く眠らず15分程度軽く寝るのがよい。
1. 解消　　2. 解決　　3. 解放　　4. 解散
> **正解：1（解消）**　「解消」＝不快な状態をなくすこと。「眠気を解消する」「ストレスを解消する」のように使う。「解決」は問題を片づける、「解放」は自由にする、「解散」は集まりを解く。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　歴史ある建物だが、古くなって危険なので（　　）。
1. 壊してしょうがない　　2. 壊さないはずだ　　3. 壊しっこない　　4. 壊さざるを得ない
> **正解：4（壊さざるを得ない）**　「〜ざるを得ない」＝したくないがやむを得ずしなければならない。「壊さざるを得ない」は仕方なく壊すという意味。「〜っこない」は絶対〜できない。

**6**　テレビ番組は内容によっては、若者に悪い影響を（　　）。
1. 与えざるを得ない　　2. 与えかねない　　3. 与えないところだ　　4. 与えっこない
> **正解：2（与えかねない）**　「〜かねない」＝（よくないことが）起こる可能性がある。「悪い影響を与えかねない」は与えてしまうかもしれないという懸念を表す。

**7**　仕事は長い時間働けばいい（　　）。時間をかけないで効率よく進めることを考える必要がある。
1. に限る　　2. ということだ　　3. というものではない　　4. に決まっている
> **正解：3（というものではない）**　「〜というものではない」＝〜とは限らない・必ずしも〜ではない。「長く働けばいいというものではない」は時間だけが重要ではないという意味。「に限る」は〜が一番。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　そうじゃないのよ。＿＿ ＿＿ <span class="star">★</span> ＿＿ 好きなのよ。
1. 雰囲気が　　2. というより　　3. 好き　　4. お酒が
> **正解：2（というより）**　正しい語順：「お酒が→好き→★というより→雰囲気が」。完成文：「お酒が好きというより雰囲気が好きなのよ。」「〜というより」＝〜というよりむしろ（より正確な言い換え）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　会社に勤めている限り、前の晩どんなに遅く帰宅しても、翌朝はいつも通り９時に出社せざるを得ないのが日本のサラリーマンだ。午前中はともかく午後２時を過ぎると、人間の体のリズムから自然に眠くなる。そうなったら、事故や仕事のミスを生みかねない。睡眠不足は、会社が休みの日にたくさん寝れば、解消するというものではない。眠くなる時間帯に短い睡眠を取るほうが、眠くなるのを防げると専門家は言う。
>
> 　東京のビルにある「仮眠室」には、多い日には120人も訪れるそうだ。利用者の話では、眠気を解消するには、深く眠らないで15分程度軽く寝るのがいいということだ。

**9**　日本のサラリーマンについて、筆者はどう述べているか。
1. 前の晩遅く帰っても、翌朝はいつも通り９時に出社する。
2. 眠くなる時間には必ず仮眠を取っている。
3. 休みの日にたくさん寝て睡眠不足を解消している。
4. 午後２時を過ぎると仕事のミスをしなくなる。
> **正解：1**　「前の晩どんなに遅く帰宅しても、翌朝はいつも通り９時に出社せざるを得ない」とある。3は「解消するというものではない」と否定され、4は逆（ミスを生みかねない）なので誤り。

**10**　眠気を解消するために、専門家は何がよいと言っているか。
1. 休みの日にまとめて長く眠ること
2. 眠くなる時間帯に15分程度の短い睡眠を取ること
3. 午前中に深く眠ること
4. 仮眠室をいっさい使わないこと
> **正解：2**　「眠くなる時間帯に短い睡眠を取るほうが…防げる」「15分程度軽く寝るのがいい」とある。休日にまとめて寝ても解消しないと述べられている。
