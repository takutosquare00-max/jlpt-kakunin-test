＿＿のことばの読み方として最もよいものを、1・2・3・4から一つ選んでください。

✅ 正解：3. じゅんび — 「準備」=「じゅんび」。preparation。

✅ 正解：1. れんらく — 「連絡」=「れんらく」。contact。

✅ 正解：1. よてい — 「予定」=「よてい」。plan。

（　）に入れるのに最もよいものを、1・2・3・4から一つ選んでください。

✅ 正解：2. のりました — 飛行機・電車・バスには「乗る」を使います。

✅ 正解：2. たいへん — 「大変ですが、楽しい」で「大変だけど楽しい」という逆接の意味。

✅ 正解：2. こんで — 「込んでいる」= 混雑している。予約が必要な理由に合う。
14さいきん　うんどうしていないので、（　）が　ふえました。

15この　カメラは　よく　（　）ので、かいました。
✅ 正解：3. うれている — 「売れている」（うれている）= 売れ行きが良い。人気があるので買った。
16しけんの　（　）を　しました。
✅ 正解：1. じゅんび — 「試験の準備」は exam preparation。準備をしました。

18レストランを　（　）　しました。

19この　みせは　（　）　あいている。
✅ 正解：2. いつも — 「いつも開いている」= always open。

もんだい3 — 文法（12問）

（　）に入れるのに最もよいものを、1・2・3・4から一つ選んでください。

21しゅくだいを　わすれて　しまった。先生に　（　）かもしれない。
✅ 正解：2. おこられる — 「先生に怒られる」は受身形。先生が怒る→自分が怒られる。
22A「日本語が　じょうずに　なりましたね。」
B「ありがとうございます。毎日　（　）ようにしています。」
✅ 正解：1. べんきょうする — 「〜ようにする」は努力・心がけを表します。辞書形＋ようにする。
23もし　100万円（　）、何が　したいですか。
✅ 正解：2. あったら — 「もし〜たら」で仮定条件を表します。「もし100万円あったら」が自然。
24じゅぎょうが　おわった（　）、しょくいんしつに　来てください。
✅ 正解：1. あとで — 「〜あとで」で after 〜。授業が終わった後に来てください。
25来週の　水曜日（　）　レポートを　出してください。
✅ 正解：1. までに — 「〜までに」で by (time)。期限を表す。
26あの　レストランは　おいしい（　）です。
✅ 正解：2. らしい — 「〜らしい」で I heard、apparently。伝聞を表す。
27日本に　来てから、日本語が　（　）ようになりました。
✅ 正解：2. 話せる — 「〜ようになる」の前は可能形。日本語が話せるようになった。
28テニスを　（　）　みませんか。
✅ 正解：2. して — 「〜てみる」で try doing。テニスをしてみませんか＝やってみませんか。
29でんきを　（　）　でかけてください。
✅ 正解：2. けして — 「電気を消して出かけて」で て形で接続。
30雨が　（　）　かさを　かりましょう。
✅ 正解：2. ふったら — 「〜たら」で条件。雨が降ったら傘を借りましょう。
31この　ほんは　やさしい（　）　おもしろいです。
✅ 正解：2. し — 「〜し〜し」で and、and what's more。やさしいしおもしろい。
32れきしの　本を　（　）　あります。
✅ 正解：1. よんで — 「読んであります」= have read (in preparation)。て形＋ある。
もんだい4 — 言い換え（5問）
33この　もんだいは　かんたんです。
✅ 正解：1 — 「簡単」≒「易しい」（やさしい）。どちらも難しくないという意味。
34あの人の　はなしは　まったく　わかりません。
✅ 正解：2. ぜんぜん — 「まったく〜ない」≒「全然〜ない」。完全否定。
35この　しごとは　ふくざつです。
✅ 正解：2 — 「複雑」は「簡単ではない」という意味に近い。
36かれは　ぜひ　日本に　行きたいと　言っています。
✅ 正解：3. 必ず — 「ぜひ」= by all means、certainly。「必ず」も同様に強く望む気持ちを表す。
37この　道は　あぶないです。
✅ 正解：2 — 「危ない」=「危険」。dangerous の意味。
もんだい5 — 文の組み立て（5問）

★に入る最もよいものを選んでください。4つの選択肢をすべて使って正しい文を作る順序で、★の位置に入るものを選んでください。

38友だちに ＿＿ ＿＿ ★ ＿＿ もらいました。
✅ 正解：2. ていねいに — 正しい語順「友だちに きのう 日本語を ★ていねいに おしえて もらいました。」4つすべてを使って並び替える。
39私に ＿＿ ＿＿ ★ ＿＿ ください。
✅ 正解：4. ていねいに — 正しい語順「私に 電車の のり方を ★ていねいに おしえて ください。」4つすべてを使って並び替える。
40来週の ＿＿ ＿＿ ★ ＿＿ あります。
✅ 正解：4. しょくいんしつで — 正しい語順「来週の 水曜日に 午後3時から しょくいんしつで 会議が あります。」4つすべてを使って並び替える。★の位置はしょくいんしつで。
41母に ＿＿ ＿＿ ★ ＿＿ あげました。
✅ 正解：1. プレゼントを — 正しい語順「母に きのう たんじょうびの ★プレゼントを 買って あげました。」4つすべてを使って並び替える。
42弟に ＿＿ ＿＿ ★ ＿＿ やりました。
✅ 正解：4. ていねいに — 正しい語順「弟に きのう 英語を ていねいに おしえて やりました。」4つすべてを使って並び替える。★の位置はていねいに。

来週の土曜日、学校で「文化祭」があります。午前10時から午後4時までです。各クラスが食べ物や飲み物を売ります。また、体育館でダンスや歌の発表があります。保護者の方も来ることができます。入場は無料です。ぜひ来てください。
