# 確認テストの作成ルール

## 1. 参照フォルダ

- 指示に応じて、各レベルのフォルダ「N1」「N2」「N3」「N4」「N5」を参照すること

## 2. 問題の使用ルール

- 「2-kakunin-test」フォルダに既に作成されている確認テストに使用している問題は**使用しない**こと
- なるべく過去に使用していない単語を使用すること
- 同じ単語を違う問題として使用することは**可**とする

## 3. 問題の作り方のルール

- そのレベル（N1〜N5）の問題を Web 検索して抜粋すること
- 各問題の種類：
  - Q1, Q2：漢字の読み
  - Q3, Q4：語彙
  - Q5, Q6, Q7：文法
  - Q8：言い換え
  - Q9：文の組み立て
  - Q10：読解

## 4. 正解のルール

- Q1からQ10（Q9を除く）回答は常に**1つのみ**が確実に正解になるようにすること
- 複数が正解になりうる選択肢は禁止
- 正解の位置を分散させること
- Q9は4つの選択肢を全て使用できるようにすること

## 5. 参考資料

- 「1.問題集」→「公式」の全ファイル情報を**問題構成の参考**にすること
- 「1.問題集」→「追加情報」の全ファイル情報を**問題作成の参考**にすること
