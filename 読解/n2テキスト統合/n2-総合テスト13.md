## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　健康にいいと言われる栄養 <u>補助</u> 食品だけを食べる人が増えている。
1. ほじょ　　2. ほすけ　　3. ぼじょ　　4. ほじょう
> **正解：1（ほじょ）**　「補助」＝足りないところを補い助けること。「補」＝ホ、「助」＝ジョ。「栄養補助食品」「補助金」のように使う。

**2**　体のためには、<u>栄養</u> のバランスを考えることが大切だ。
1. えいよう　　2. えよう　　3. えいよ　　4. えいゆう
> **正解：1（えいよう）**　「栄養」＝体を保ち成長させるために必要な成分。「栄」＝エイ、「養」＝ヨウ。「栄養のバランス」「栄養がある」のように使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　体にいいものを食べていると思っているので、本人にはまったく（　　）感がない。
1. 危機　　2. 危険　　3. 危害　　4. 危急
> **正解：1（危機）**　「危機感」＝危ない状態だと感じる気持ち。「危機感がない」のように使う慣用的な語。「危険」は危ない状態そのもの、「危害」は害を加えること。

**4**　「体にいい」と（　　）て、そればかり食べていると、健康を損なうことがある。
1. 信じ　　2. 感じ　　3. 応じ　　4. 通じ
> **正解：1（信じ）**　「信じる」＝本当だと思い込む。「体にいいと信じて食べる」のように使う。「感じる」は心や体で受け取る、「応じる」は相手に合わせる、「通じる」は伝わる・つながる。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　甘いものはつい食べすぎてしまい（　　）ですが、健康を考えるとあまり食べないほうがいいでしょう。
1. がち　　2. げ　　3. しだい　　4. たい
> **正解：1（がち）**　「〜がち」＝〜しやすい・〜になりがち（よくない傾向）。「食べすぎてしまいがち」はついつい食べすぎてしまう傾向があるという意味。「〜げ」は〜そうな様子。

**6**　工場建設（　　）、部長は打ち合わせのためにロシアへ向かった。
1. において　　2. に先立って　　3. にわたって　　4. につけ
> **正解：2（に先立って）**　「〜に先立って」＝〜に先行して・〜の前に。「工場建設に先立って打ち合わせに行った」は建設開始の前に準備のため行ったという意味。

**7**　私はケーキ作り（　　）誰にも負けないと思っている。
1. にともなって　　2. に先立って　　3. にしたがって　　4. にかけては
> **正解：4（にかけては）**　「〜にかけては」＝〜の点に関しては（特定分野での自信）。「ケーキ作りにかけては誰にも負けない」はケーキ作りの腕には自信があるという意味。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　有給休暇は ＿＿ ＿＿ <span class="star">★</span> ＿＿ 申請してください。
1. 上司の　　2. 得た　　3. 上で　　4. 許可を
> **正解：3（上で）**　正しい語順：「上司の→許可を→得た→★上で」。完成文：「有給休暇は上司の許可を得た上で申請してください。」「〜た上で」＝〜してから次のことをするという手順の表現。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　最近、健康にいいと言われる食品や栄養補助食品だけを食べる「偏食症」が急増しています。本人は体にいいものを食べていると思っているので、まったく危機感がないのが特徴です。野菜ジュースや納豆は「体にいい」から、それさえ食べれば健康に生活できると思いがちです。でも「体にいい」と信じてそれだけを食べているばかりでは、大きな間違いです。
>
> 　健康を気にする人のニーズにこたえて様々な栄養補助食品が売られたりしていることもその原因の１つかもしれませんが、栄養のバランスを考えた上で本当に体によい食事をとるようにしましょう。

**9**　「偏食症」の人の特徴は何か。
1. 体に悪いものばかりを好んで食べる。
2. 体にいいものを食べていると思っていて、危機感がない。
3. 何も食べようとしない。
4. 一度に大量に食べてしまう。
> **正解：2**　「本人は体にいいものを食べていると思っているので、まったく危機感がないのが特徴です」とある。よいものだと信じているため問題意識がない。

**10**　筆者が勧めていることは何か。
1. 健康食品だけを食べること
2. 栄養のバランスを考えて、本当に体によい食事をとること
3. 野菜ジュースだけを飲むこと
4. できるだけ食事をしないこと
> **正解：2**　「栄養のバランスを考えた上で本当に体によい食事をとるようにしましょう」とある。特定の食品に偏らずバランスのよい食事を勧めている。
