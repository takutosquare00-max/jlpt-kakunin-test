# N2文法 ランダムテスト No.12（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　郊外に新しくできたスーパーに行ってみたいと思っている（　　）、車がないから、行きようがない。
1. あげく　　2. とともに　　3. にしては　　4. ものの

**問2**　今回の遺跡の発掘によって、古代文明の謎が明らかになり（　　）。
1. つつある　　2. かねない　　3. 得ない　　4. っぽい

**問3**　この靴はデザイン（　　）たいへん歩きやすいので気に入っています。
1. として　　2. について　　3. によって　　4. はともかく

**問4**　B：30分でもあるだけ（　　）よ。
1. といった　　2. ましだ　　3. というものだ　　4. らしい

**問5**　成績がトップだった彼は、今回のテストで2位になって人生が終わってしまった（　　）落ち込んでいる。
1. かと思うと　　2. かのように　　3. 限り　　4. わりに

**問6**　準備が整い（　　）、始めさせていただきますので、そのまま少々お待ちください。
1. ながら　　2. そうに　　3. 次第　　4. かねないので

**問7**　もし明日死ぬ（　　）今日１日何をして過ごしたいですか。
1. としたら　　2. ことなく　　3. 上で　　4. ものの

**問8**　リムジンバスが渋滞で遅れて、もう少しで飛行機に乗り遅れる（　　）。
1. ことはないだろう　　2. ところだった　　3. ところではなかった　　4. ばかりだった

**問9**　この機会にぜひ ＿＿ ＿＿ ★ ＿＿ 申し上げます。
1. ご案内　　2. お試し　　3. 新商品を　　4. くださいますよう

**問10**　私は今の仕事が不満な（　　）けど、もっとおもしろい仕事がしたいと思う。
1. つもりだ　　2. ようだ　　3. はずがない　　4. わけじゃない

---

## 解答と解説

**問1**　正解：**4**（ものの）
「〜ものの」＝〜だけれど（逆接）。「行きたいと思っているものの」は行きたいとは思うがという意味。「あげく」は〜した末に（よくない結果）、「にしては」は〜の割に（期待外れ）。

**問2**　正解：**1**（つつある）
「〜つつある」＝変化がだんだん進行中。「明らかになりつつある」はどんどん明らかになってきているという意味。「かねない」は〜の恐れがある、「っぽい」は〜っぽい（傾向・感じ）。

**問3**　正解：**4**（はともかく）
「〜はともかく」＝〜はさておき・〜はいったん置いて。「デザインはともかく歩きやすい」はデザインの話は別にして機能性がいいという意味。

**問4**　正解：**2**（ましだ）
「〜ましだ」＝〜の方がまだいい（比較）。「あるだけましだ」はないよりはいいという意味。「〜よりましだ」という形でよく使われる。「というものだ」は一般論・感慨、「らしい」は推量。

**問5**　正解：**2**（かのように）
「〜かのように」＝まるで〜のように（実際とは違う比喩）。「人生が終わったかのように落ち込んでいる」は実際には終わっていないのに、まるでそのような状態だという意味。

**問6**　正解：**3**（次第）
「〜次第」＝〜したらすぐに（動詞ます形に接続）。「整い次第、始めます」は準備ができたらすぐ始めるという意味。「次第」は完了後すぐの行動を表すので「お待ちください」と自然につながる。

**問7**　正解：**1**（としたら）
「〜としたら」＝〜と仮定したら（仮定条件）。「明日死ぬとしたら何をしたいか」は仮の話として何を選ぶかという意味。「ものの」は〜だけれど（逆接）、「上で」は〜した上で（手順）。

**問8**　正解：**2**（ところだった）
「〜ところだった」＝〜しそうだった（が実際はしなかった）。「乗り遅れるところだった」は危うく乗り遅れそうだったという意味。「もう少しで〜ところだった」という形でよく使う。

**問9**　正解：**4**（くださいますよう）
正しい語順：「新商品を→お試し→★くださいますよう→ご案内」。完成文：「ぜひ新商品をお試しくださいますようご案内申し上げます。」「〜くださいますよう」＝〜してほしいという丁寧な依頼。

**問10**　正解：**4**（わけじゃない）
「〜わけじゃない」＝必ずしも〜ではない・〜というわけでもない。「不満なわけじゃないけど」は完全に不満とは言えないけれど、という婉曲表現。
