# N2文法 ランダムテスト No.10（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　食べ物を売る店は、どんな店（　　）衛生管理をきちんとしなければならない。
1. にしては　　2. にしろ　　3. につけ　　4. ばかりか

**問2**　課長は忙しい ＿＿ ＿＿ ★ ＿＿ 頼んでくるから困るんだ。
1. に限って　　2. めんどうな　　3. とき　　4. 仕事を

**問3**　この本は子どものころ一度読んだ（　　）、ストーリーも忘れてしまいました。
1. ことに　　2. きりで　　3. あげく　　4. 限り

**問4**　みんなが ＿＿ ＿＿ ★ ＿＿ 気になる。
1. 話か　　2. 楽しげに　　3. 話していると　　4. 何の

**問5**　本日はグラフィックデザイナー（　　）ご活躍の渡辺たかしさんにお話を伺いたいと思います。
1. として　　2. を問わず　　3. をはじめ　　4. はもとより

**問6**　同じ失敗はくり返す（　　）と思っていても、ついやってしまうのが人間だ。
1. ことだ　　2. にすぎない　　3. まい　　4. まいか

**問7**　駅では ＿＿ ＿＿ ★ ＿＿ 数を少なくしている。
1. 上の　　2. ごみ箱の　　3. 防犯　　4. 理由から

**問8**　学生時代はよくこの公園の芝生で昼寝をした（　　）。懐かしいなあ。
1. ことだ　　2. ものだ　　3. だけだ　　4. わけだ

**問9**　B：もちろんだよ。まじめに ＿＿ ＿＿ ★ ＿＿ からね。
1. クビに　　2. やらないと　　3. かねない　　4. され

**問10**　先週から今週（　　）、ホームページのアクセス数が急増して驚いた。
1. にともない　　2. にかけて　　3. にとって　　4. とともに

---

## 解答と解説

**問1**　正解：**2**（にしろ）
「〜にしろ」＝〜であっても・〜にしても（譲歩）。「どんな店にしろ衛生管理は必要」はどのような店であっても同じという意味。「にしては」は〜の割に（期待外れ）、「につけ」は〜するたびに。

**問2**　正解：**2**（めんどうな）
正しい語順：「とき→に限って→★めんどうな→仕事を」。完成文：「課長は忙しいときに限って、めんどうな仕事を頼んでくるから困るんだ。」「〜に限って」＝特定の（皮肉な）状況にだけ（限定の強調）。

**問3**　正解：**2**（きりで）
「〜きりで」＝〜だけで（それ以後何もしていない）。「一度読んだきりで」は一回読んだだけで以来読んでいない、という意味。「あげく」は〜した末に（悪い結果）を表す。

**問4**　正解：**4**（何の）
正しい語順：「楽しげに→話していると→★何の→話か」。完成文：「みんなが楽しげに話していると、何の話か気になる。」「〜か気になる」＝間接疑問文（何の話かを知りたい）の自然な形。

**問5**　正解：**1**（として）
「〜として」＝〜という立場・資格・役割で。「デザイナーとして活躍している」はデザイナーという職業で活躍している、という意味。「をはじめ」は〜を代表として（複数のものを挙げる）。

**問6**　正解：**3**（まい）
「〜まい」＝〜しないようにしよう・〜しまい（否定の意志・推量）。「くり返すまいと思っていても」はくり返さないようにしようと思っていても、という意味。「にすぎない」は〜でしかない（過小評価）。

**問7**　正解：**4**（理由から）
正しい語順：「防犯→上の→★理由から→ごみ箱の」。完成文：「駅では防犯上の理由からごみ箱の数を少なくしている。」「〜上の〜から」＝〜の観点から・〜の事情によって。

**問8**　正解：**2**（ものだ）
「〜ものだ」＝過去の習慣・回想（懐かしい気持ち）。「よくした/したものだ」は昔よくそうしていたという懐かしさを表す。「ことだ」は感慨・感嘆、「わけだ」は理由や当然の結論を表す。

**問9**　正解：**4**（され）
正しい語順：「やらないと→クビに→され→★かねない」。完成文：「まじめにやらないとクビにされかねないからね。」「〜かねない」＝〜してしまう可能性がある（悪い結果への警告）。

**問10**　正解：**2**（にかけて）
「〜から〜にかけて」＝ある期間・範囲にわたって。「先週から今週にかけて」は先週〜今週の間という意味。「にともなって」は原因と変化の関係を表し、期間の幅には使えない。
