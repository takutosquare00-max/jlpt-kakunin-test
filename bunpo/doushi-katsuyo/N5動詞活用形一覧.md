# N5動詞 活用形一覧

動詞の後ろに来る言葉ごとに、動詞がどのように変化するかを示しています。

## 後ろの言葉と活用形

| 後ろの言葉 | 活用形 |
| :--- | :--- |
| ない, よう, (う) | 未然形 |
| ます, た | 連用形 |
| と, 。 | 終止形 |
| の, こと | 連体形 |
| ば | 仮定形 |
| 。（文末） | 命令形 |
