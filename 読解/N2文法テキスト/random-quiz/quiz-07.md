# N2文法 ランダムテスト No.07（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　先生は新内閣についてどう（　　）か。
1. 存じます　　2. おわかりです　　3. お考えです　　4. 存じ上げます

**問2**　上手にしからないと ＿＿ ＿＿ ★ ＿＿ プレッシャーでいい面もつぶしてしまいかねません。
1. 能力を　　2. どころか　　3. 引き出す　　4. 部下の

**問3**　この町は大学（　　）、若者向けの商店やアパートが集まっている。
1. を問わず　　2. を通じて　　3. を中心として　　4. をもととして

**問4**　このドキュメンタリーは南極の厳しい自然の中を生き（　　）２匹の犬の物語である。
1. ぬいた　　2. がたい　　3. かねない　　4. 得る

**問5**　話し上手になるには、 ＿＿ ＿＿ ★ ＿＿ 話をよく聞き、情報を得て、話題を増やしましょう。
1. 人の　　2. なった　　3. レポーターに　　4. つもりで

**問6**　インターンシップには参加したほうが就職に有利だと言われるが、＿＿ ＿＿ ★ ＿＿ わけではない。
1. 内定が　　2. もらえる　　3. という　　4. 参加すれば

**問7**　宇宙飛行士を ＿＿ ＿＿ ★ ＿＿ 判断力や問題解決能力もきたえておかなければならない。
1. 体力　　2. はもとより　　3. からには　　4. 目指す

**問8**　新製品の ＿＿ ＿＿ ★ ＿＿ アンケート結果は重要な資料になるだろう。
1. 上で　　2. この　　3. 考える　　4. 宣伝方法を

**問9**　面接のチャンス（　　）もらえれば、私の熱意が伝えられるのに…。
1. ながら　　2. に限り　　3. さえ　　4. を問わず

**問10**　試合会場へのペットボトルの持ち込みが禁止になったのは、（　　）グラウンドに物を投げるファンが増えたからです。
1. 興奮することなく　　2. 興奮するどころか　　3. 興奮するからには　　4. 興奮のあまり

---

## 解答と解説

**問1**　正解：**3**（お考えです）
「お考えです」＝「考えている」の尊敬語。先生の考えを尋ねているので尊敬語が必要。「存じます」は「思う」の謙譲語（自分の考え）、「存じ上げます」は人に対して使う謙譲語。

**問2**　正解：**3**（引き出す）
正しい語順：「部下の→能力を→★引き出す→どころか」。完成文：「上手にしからないと部下の能力を引き出すどころか、プレッシャーでいい面もつぶしてしまいかねません。」「〜どころか」＝〜は全くなく、むしろ逆。

**問3**　正解：**3**（を中心として）
「〜を中心として」＝〜を中心に。「大学を中心として」は大学の周辺に、という意味。「を通じて」はずっと・全体を通して、「を問わず」は〜に関係なく、を表す。

**問4**　正解：**1**（ぬいた）
「〜ぬく」＝最後まで〜する・〜し切る（完遂・努力の継続）。「生きぬいた」は厳しい環境を最後まで生き続けたという意味。「がたい」は〜しにくい（困難）、「かねない」は〜の恐れがある（悪い可能性）。

**問5**　正解：**4**（つもりで）
正しい語順：「レポーターに→なった→★つもりで→人の」。完成文：「レポーターになったつもりで人の話をよく聞き、情報を得て、話題を増やしましょう。」「〜たつもりで」＝〜と仮定して・〜と想像して。

**問6**　正解：**2**（もらえる）
正しい語順：「参加すれば→内定が→★もらえる→という」。完成文：「参加すれば内定がもらえるというわけではない。」「〜というわけではない」＝必ずしも〜ではない（断定を避ける）。

**問7**　正解：**1**（体力）
正しい語順：「目指す→からには→★体力→はもとより」。完成文：「宇宙飛行士を目指すからには、体力はもとより判断力や問題解決能力もきたえておかなければならない。」「〜はもとより」＝〜はもちろん。

**問8**　正解：**1**（上で）
正しい語順：「宣伝方法を→考える→★上で→この」。完成文：「新製品の宣伝方法を考える上で、このアンケート結果は重要な資料になるだろう。」「〜する上で」＝〜をするときに・〜するにあたって。

**問9**　正解：**3**（さえ）
「〜さえ〜ば」＝〜だけで十分・〜という条件が整えば。「チャンスさえもらえれば」はそれだけあればという意味。「〜に限り」は〜だけという限定、「ながら」は〜しながらという同時進行を表す。

**問10**　正解：**4**（興奮のあまり）
「〜のあまり」＝〜が強すぎて・〜しすぎて（感情・状態が原因）。「興奮のあまり物を投げる」は興奮しすぎてそういう行動になるという意味。「〜することなく」は〜せずに（否定）。
