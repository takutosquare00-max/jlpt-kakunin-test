## IV. หลักไวยากรณ์

### 1. ～場合は、～

```
คำกริยารูปปกจานุกรม
คำกริยารูป ない（〜ない）    ┐
คำกริยารูป た               ├ 場合は、〜
คำคุณศัพท์ い（〜い）       │
คำคุณศัพท์ な（〜な）       │
คำนาม の                    ┘
```

〜ばあい เป็นส่วนแสดงการสมมติสถานการณ์ โดยใดสถานการณ์หนึ่ง ประโยคหลังจะแสดงวิธีการแก้ปัญหาหรือผลที่เกิดขึ้น ばあい ถือเป็นคำนาม จึงมีวิธีเชื่อมคำเหมือนการขยายคำนาม

① 会議に 間に 合わない **場合は**、連絡して ください。
   กรณีที่จะมาไม่ทันการประชุม กรุณาติดต่อมาด้วย

② 時間に 遅れた **場合は**、会場に 入れません。
   กรณีที่มาช้ากว่าเวลา จะไม่สามารถเข้าไปในสถานที่จัดงานได้

③ パソコンの 調子が 悪い **場合は**、どう したら いいですか。
   กรณีที่คอมพิวเตอร์อาการไม่ดี ควรทำอย่างไร

④ 領収書が 必要な **場合は**、言って ください。
   กรณีจำเป็นต้องใช้ใบเสร็จ กรุณาแจ้ง

⑤ 火事や 地震の **場合は**、エレベーターを 使わないで ください。
   กรณีเกิดเพลิงไหม้หรือแผ่นดินไหว กรุณาอย่าใช้ลิฟต์โดยเด็ดขาด

### 2. ～のに、～

```
คำกริยา          ┐ รูปธรรมดา
คำคุณศัพท์ い    ├              のに、〜
คำคุณศัพท์ な    │ รูปธรรมดา
คำนาม            ┘ 〜だ→〜な
```

のに ใช้ในกรณีที่ประโยคหลังไม่เป็นไปตามที่คาดการณ์หวังเอาไว้ในประโยคแรก ส่วนใหญ่จะแสดงความรู้สึกผิดความคาดหมายหรือไม่พอใจ

⑥ 約束を したのに、彼女は 来ませんでした。
   ทั้งๆ ที่นัดกันแล้ว แต่เธอก็ไม่มา

⑦ きょうは 日曜日なのに、働かなければ なりません。
   ทั้งๆ ที่วันนี้เป็นวันอาทิตย์ แต่ต้องไปทำงาน

ประโยคตัวอย่างข้อ ⑥ จากประโยคแรกผู้พูดคาดหวังว่า "เธอจะมาตามที่ได้นัดกันไว้" จึงรู้สึกเสียใจและผิดคาดที่เธอไม่มา ส่วนประโยคตัวอย่างข้อ ⑦ จากประโยคแรกผู้พูดคาดหวังว่า "จะได้หยุดเนื่องจากเป็นวันอาทิตย์" แต่ก็ต้องไปทำงาน จึงใช้ のに เพื่อแสดงความรู้สึกไม่พอใจ

**[หมายเหตุ 1] ความแตกต่างระหว่าง 〜のに กับ 〜が：**

เมื่อแทนที่ 〜のに ในข้อ ⑥ และ ⑦ ด้วย 〜が จะไม่แสดงความรู้สึกผิดความคาดหมายหรือไม่พอใจ

⑧ 約束を しましたが、彼女は 来ませんでした。
   นัดกันแล้ว แต่เธอก็ไม่มา

⑨ きょうは 日曜日ですが、働かなければ なりません。
   วันนี้เป็นวันอาทิตย์ แต่ต้องไปทำงาน

**[หมายเหตุ 2] ความแตกต่างระหว่าง 〜のに กับ 〜ても：**

〜のに แสดงความรู้สึกของผู้พูดเกี่ยวกับสิ่งที่เกิดขึ้นแล้ว ไม่สามารถแสดงเหตุการณ์ที่เกิดขึ้นภายใต้เงื่อนไขสมมติในทางตรงกันข้ามกับที่คาดการณ์เอาไว้เหมือนกับ 〜ても

⑩ あした 雨が 降っても、サッカーを します。
   พรุ่งนี้ถึงฝนจะตกก็จะเล่นฟุตบอล

   × あした 雨が 降るのに、サッカーを します。
