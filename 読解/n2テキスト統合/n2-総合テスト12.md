## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　先日、ある企業コンサルティング会社の <u>研修</u> に参加した。
1. けんしゅう　　2. けんしゅ　　3. げんしゅう　　4. けんじゅう
> **正解：1（けんしゅう）**　「研修」＝仕事に必要な知識・技能を学ぶこと。「研」＝ケン、「修」＝シュウ。「新人研修」「研修会」のように使う。

**2**　おやつを口にすると、初対面の <u>緊張</u> がほぐれた。
1. きんちょう　　2. ぎんちょう　　3. きんちょ　　4. こんちょう
> **正解：1（きんちょう）**　「緊張」＝心や体が張りつめること。「緊」＝キン、「張」＝チョウ。「緊張がほぐれる」「緊張する」のように使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　主催者側の人に（　　）られて、しかたなく私もキャンディを口に入れた。
1. 勧め　　2. 進め　　3. 責め　　4. 攻め
> **正解：1（勧め）**　「勧める」＝相手にあることをするよう働きかける。「キャンディを勧められる」のように使う。「進める」は物事を先へ動かす、「責める」は非難する、「攻める」は攻撃する。

**4**　「おやつ」には、体にも心にもいい（　　）があることがわかった。
1. 効果　　2. 成果　　3. 結果　　4. 効力
> **正解：1（効果）**　「効果」＝あることをして現れるよい結果・ききめ。「いい効果がある」のように使う。「成果」は努力して得た結果、「結果」は単に物事の終わりの状態。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　私の町は気候が温暖な（　　）、海の幸も山の幸も豊富で暮らしやすい。
1. のに　　2. 上は　　3. 以上　　4. 上に
> **正解：4（上に）**　「〜上に」＝〜だけでなく、それに加えて（好ましい追加）。「温暖な上に食材も豊富」は気候がいいうえに食材も恵まれているという意味。「のに」は逆接。

**6**　このりんごは生産者の皆さんが自慢する（　　）、味も香りも抜群ですね。
1. にしても　　2. といっても　　3. だけあって　　4. 上に
> **正解：3（だけあって）**　「〜だけあって」＝〜だから当然・〜だけのことはある。「自慢するだけあって味も香りも抜群」は自慢するだけの理由があるという意味。

**7**　外国語を勉強するなら、その国に留学する（　　）と私は思う。
1. に限る　　2. にほかならない　　3. にすぎない　　4. ばかりだ
> **正解：1（に限る）**　「〜に限る」＝〜が最善・〜が一番いい（強い推奨）。「留学するに限る」は留学するのが最もいいという意見。「にほかならない」は〜に他ならない（断定）。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　彼女は今 ＿＿ ＿＿ <span class="star">★</span> ＿＿ テレビにも出ている。
1. よく　　2. バイオリニスト　　3. 注目の　　4. だけあって
> **正解：4（だけあって）**　正しい語順：「注目の→バイオリニスト→★だけあって→よく」。完成文：「彼女は今注目のバイオリニストだけあって、よくテレビにも出ている。」「〜だけあって」＝〜だから当然（評判通り）。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　先日、ある企業コンサルティング会社の研修に参加した。研修の初めにいきなりキャンディが配られた。確かに今回の研修はお菓子業界に関係があるものだし、女性の参加者も多い。時間も少しお腹が空いてくる「おやつ」の時間だ。しかし、まじめな気持ちで研修に参加している私は、キャンディにしろチョコレートにしろ、大人がこのような場で口にするものではないと少々不快に思った。
>
> 　ほかの参加者を見ると、女性ばかりでなく、男性も口に入れている。主催者側の人に勧められ、しかたなく私もキャンディを口に入れた。すると、疲れてボーッとしていた頭がすっきりした。周りを見ると、初対面の緊張がほぐれたのか、知らない人同士が和やかに話を始めているではないか。
>
> 　それを見て、この研修で「おやつ」が出されている意味がわかった。最近話題の企業コンサルティング会社の研修だけあって、「おやつ」をうまく取り入れていると思った。

**9**　企業コンサルティングの会社は、研修でどうして「おやつ」を出すのだと筆者は考えたか。
1. お腹が空く時間に行われるから
2. 甘いものが好きな女性の参加者が多いから
3. 体にも心にもいい効果があるから
4. お菓子業界の宣伝の意味もある研修だから
> **正解：3**　おやつを口にすると「頭がすっきりし」、「初対面の緊張がほぐれ」て参加者が和やかに話し始めた。心身によい効果があるために出されていると筆者は理解した。

**10**　「おやつ」を口にした後、参加者の様子はどう変わったか。
1. 頭がすっきりし、知らない人同士が和やかに話し始めた。
2. かえって眠くなってしまった。
3. 研修に集中できなくなった。
4. みんな不快に感じた。
> **正解：1**　「疲れてボーッとしていた頭がすっきりした」「初対面の緊張がほぐれたのか、知らない人同士が和やかに話を始めている」とある。
