# N2文法 ランダムテスト No.01（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　グレープフルーツは木になっている様子がブドウに似ている（　　）、その名前がついたそうだ。
1. のみならず　　2. ことから　　3. にもかかわらず　　4. ものの

**問2**　ブレーキの音が聞こえた（　　）、突然トラックが店に飛び込んできた。
1. かと思ったら　　2. かのように　　3. といっても　　4. からといって

**問3**　有給休暇は ＿＿ ＿＿ ★ ＿＿ 申請してください。
1. 上司の　　2. 得た　　3. 上で　　4. 許可を

**問4**　お茶のいれ方など、 ＿＿ ＿＿ ★ ＿＿ と思っていたが、きちんとした方法は意外に知らない人が多くて驚いた。
1. 人は　　2. まい　　3. いる　　4. 知らない

**問5**　私はケーキ作り（　　）誰にも負けないと思っている。
1. にともなって　　2. に先立って　　3. にしたがって　　4. にかけては

**問6**　先生から貴重な資料を（　　）ことは、私の研究の大きな助けになりました。
1. 拝見できました　　2. 拝借できました　　3. お貸しになりました　　4. お見えになりました

**問7**　ご注文を（　　）。ご利用ありがとうございます。
1. いたしました　　2. うけたまわりました　　3. なさいました　　4. されました

**問8**　すばらしい芸術は、いつ ＿＿ ＿＿ ★ ＿＿、人々を感動させる。
1. も　　2. において　　3. 時代　　4. の

**問9**　彼は ＿＿ ＿＿ ★ ＿＿ パーティーでは人気者だ。
1. 歌えば　　2. ダンスも　　3. できるので　　4. 歌も

**問10**　地域住民が新しいホテルの ＿＿ ＿＿ ★ ＿＿ 起こしたそうだ。
1. 運動を　　2. 建設　　3. 反対　　4. をめぐって

---

## 解答と解説

**問1**　正解：**2**（ことから）
「〜ことから」＝〜という理由・根拠から。「ブドウに似ていることから名前がついた」は似ているという事実がその名前の由来になったという意味。「のみならず」は〜だけでなく、を表す。

**問2**　正解：**1**（かと思ったら）
「〜かと思ったら」＝〜したと思った途端に（すぐ次のことが起きる）。「音が聞こえたかと思ったらトラックが来た」は聞こえた瞬間に入ってきたという意味。「かのように」はまるで〜のように（比喩）。

**問3**　正解：**3**（上で）
正しい語順：「上司の→許可を→得た→★上で」。完成文：「有給休暇は上司の許可を得た上で申請してください。」「〜た上で」＝〜してから次のことをするという手順の表現。

**問4**　正解：**3**（いる）
正しい語順：「知らない→人は→★いる→まい」。完成文：「知らない人はいるまいと思っていたが、意外に多くて驚いた。」「〜まい」＝〜ないだろう（否定の推量・確信）。

**問5**　正解：**4**（にかけては）
「〜にかけては」＝〜の点に関しては（特定分野での自信）。「ケーキ作りにかけては誰にも負けない」はケーキ作りの腕には自信があるという意味。「に先立って」は〜の前に（準備・手順）。

**問6**　正解：**2**（拝借できました）
「拝借する」＝「借りる」の謙譲語（自分が借りる）。先生から借りた（自分の行為）なので謙譲語。「拝見する」は見る（自分）の謙譲語、「お貸しになる」は貸す（相手）の尊敬語。

**問7**　正解：**2**（うけたまわりました）
「うけたまわる」＝「聞く・受ける」の謙譲語。自分が相手の注文を受ける行為に使う。「いたす」は「する」の謙譲語、「なさる」は「する」の尊敬語。相手の注文を受けるのは謙譲語が正しい。

**問8**　正解：**2**（において）
正しい語順：「の→時代→★において→も」。完成文：「いつの時代においても人々を感動させる。」「〜において」＝〜で（場所・時間・場面）。「いつの時代においても」はどんな時代でもという意味。

**問9**　正解：**2**（ダンスも）
正しい語順：「歌も→歌えば→★ダンスも→できるので」。完成文：「彼は歌も歌えばダンスもできるので、パーティーでは人気者だ。」「〜も〜ば〜も」＝〜もできるし〜もできる（複数の能力を並べる）。

**問10**　正解：**3**（反対）
正しい語順：「建設→をめぐって→★反対→運動を」。完成文：「地域住民が新しいホテルの建設をめぐって、反対運動を起こしたそうだ。」「〜をめぐって」＝〜を争点・話題の中心にして。
