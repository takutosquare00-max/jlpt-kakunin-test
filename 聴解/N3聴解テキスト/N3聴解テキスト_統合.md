## 3　音の高低や長さに注意しよう
## （イントネーションや音の長短に注意する）　（p.29）

イントネーションを聞くと、話す人が相手の話に賛成しているかどうかがわかることがあります。

### 表現

- 賛成しない・断る・残念な気持ち：
  - うーん／あー／えー
  - ○○ねえ／○○かー／○○ですかあ？

**例1：**
- これにしたらどう？（これにしたらどうですか）
  - ーうん、そうだね。（いいですね）
  - ーうーん、そうだねえ。（賛成していない）

**例2：**
- あした 7時に来てくれませんか。（明日7時に来てくれませんか）
  - ーあ、7時ですか。（7時ですね）
  - ーあー、7時ですかー。（賛成していない）
  - ーえー、7時ですかあ？（賛成していない）

## II　「言葉の選び方」のスキルを学ぼう
## （「言葉の選び方」のスキルを学ぼう）

### 問題の形と内容　（p.31）

絵を見ながら、状況を説明する文と質問を聞きます。それから3つの選択肢を聞いて、矢印（→）の人が言う言葉として一番よいものを選びます。

**流れ：**
絵を見ながら状況説明文と質問を聞く　→　3つの選択肢を聞く　→　答えを選ぶ

**◇ 聞くときのポイント：**
1. 状況説明文を理解する
2. その表現が「話す人の動作」か「聞く人の動作」かに注意する
3. その場面で使う特別な表現に注意する

---

### 1　状況説明文を聞く　（p.32）

状況説明文を聞いて、どんな場面かを理解しましょう。大切なのは、その動作を話す人（矢印の人）がするのか、聞く人がするのかを考えることです。

**◇ 状況説明文の例：**

**【話す人がする】**
- 例1：友達の消しゴムを使いたいです。（話す人が使う）→ 友達の消しゴムを使いたいです
- 例2：友達が疲れたので、運転を代わってあげます。（話す人が運転する）→ 友達の代わりに運転します

**【聞く人がする】**
- 例3：友達に自転車を貸してもらいます。（聞く人が貸す）→ 友達に自転車を貸してもらいます
- 例4：先生に作文の間違いを直してほしいです。（聞く人が直す）→ 先生に作文の間違いを直してもらいたいです

### 2-B　注意する動詞　（p.35）

同じ場面でも、使う動詞が違うことがあります。話す人の動作を表す動詞（例：借りる）と、聞く人の動作を表す動詞（例：貸す）では、使う表現が違います。注意しましょう。

| 動詞 | 例文 |
|---------|----------------|
| 借りる（借りる）【話す人の動作】 | 借りてもよろしいですか。**借りてもいいですか** |
| 貸す（貸す）【聞く人の動作】 | 貸していただけませんか。**貸してもらえませんか** |
| 見る（見る）【話す人の動作】 | 見てもいい? **見てもいいですか** |
| 見せる（見せる）【聞く人の動作】 | 見せてよ。**見せてください** |
| 聞く（聞く・質問する）【話す人の動作】 | お聞きしたいんですが。**質問したいです** |
| 教える（教える）【聞く人の動作】 | 教えていただきたいんですが。**教えてもらいたいです** |
| もらう／いただく（もらう）【話す人の動作】 | もらってもいいでしょうか。**もらってもいいですか** |
| くれる／くださる（くれる）【聞く人の動作】 | くださいませんか。**くれませんか** |
| 預ける（預ける）【話す人の動作】 | 荷物、預けていただけませんか。**荷物を預けてもいいですか** |
| 預かる（預かる）【聞く人の動作】 | 荷物、預かっていただけませんか。**荷物を預かってもらえませんか** |

### 1　だれの動作かに注意しよう

---

### 1-A　敬語　（p.42）

敬語を使う会話では、敬語の意味と、だれの動作かに注意しましょう。

**【一般的な形】**

**聞く人がする（尊敬語）：**

| 形 | 例 | 意味 |
|------|--------|--------|
| お ＋ 動詞のます形の語幹 ＋ になる | お使いになりますか。／お聞きになりませんか。 | 使いますか／聞きませんか |
| お ＋ 動詞のます形の語幹 ＋ ください | お待ちください。／おかけください。 | 待ってください／座ってください |
| お ＋ 動詞のます形の語幹 ＋ だ | お読みですか。／お帰りですか。 | 読みますか／帰りますか |
| 〜（ら）れる | 来られますか。／準備されましたか。 | 来ますか／準備しましたか |
| 〜ていただく | 書いていただけませんか。（書く人 = 聞く人） | 書いてもらえませんか |

**話す人がする（謙譲語）：**

| 形 | 例 | 意味 |
|------|--------|--------|
| お ＋ 動詞のます形の語幹 ＋ する | お持ちしましょう。／お渡しします。 | 持ちましょうか／渡します |
| 〜（さ）せていただく | 休ませていただけませんか。（休む人 = 話す人） | 休ませてもらえませんか |

---

**【特別な形】**　（p.43）

| 普通の動詞 | 意味 | 聞く人がする（尊敬語） | 話す人がする（謙譲語） |
|----------------|---------|---------------------|---------------------|
| する | する | なさいます | **いたします** |
| いる | いる | いらっしゃいます | おります |
| 行く／来る | 行く／来る | おいでになります | **まいります**・うかがいます |
| 聞く | 聞く・質問する | （お聞きになります） | うかがいます |
| 見る | 見る | ご覧になります | **拝見します** |
| 言う | 言う | おっしゃいます | 申し上げます |
| 食べる／飲む | 食べる／飲む | 召し上がります | いただきます |
| もらう | もらう | （お取りになります など） | いただきます |
| くれる・あげる | くれる・あげる | くださいます | さしあげます |
| 知っている | 知っている | ご存知です | 存じています |
| 会う | 会う | （お会いになります） | **お目にかかります** |

**ていねいな会話では、次のような質問を使います。注意しましょう：**
- どちらから（＝どこから）いらっしゃいましたか。（どこから来ましたか）
- どちら様／どなた（＝だれ）でしょうか。（だれですか）
- 何名様（＝何人）ですか。（何人ですか）
