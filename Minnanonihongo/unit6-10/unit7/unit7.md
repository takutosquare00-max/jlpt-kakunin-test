## 練習B

**162** 1. 例：ごはんを　食べます → はしで　ごはんを　食べます。
- 1) 手紙を　書きます →
- 2) 紙を　切ります →
- 3) ごはんを　食べます →
- 4) 写真を　撮ります →

（絵：例＝はし、1）ペン、2）はさみ、3）ナイフとフォーク、4）カメラ）

**163** 2. 例：→ これは　日本語で　何ですか。……「パソコン」です。
- 1) →
- 2) →
- 3) →
- 4) →

（絵：例＝パソコン、1）ホッチキス、2）テープ、3）ホチキス、4）電卓）

**164** 3. 例：あげます → テレーザちゃんに　ノートを　あげます。
- 1) 貸します →
- 2) 教えます →
- 3) 書きます →
- 4) かけます →

（絵：例＝テレーザちゃん、1）イーさん、2）太郎ちゃん、3）母、4）シュミットさん）

**165** 4. 例：もらいます → 父に　シャツを　もらいました。
- 1) 借ります →
- 2) 習います →
- 3) もらいます →
- 4) もらいます →

（絵：例＝父、1）カリナさん、2）ワンさん、3）木村さん（メール）、4）サントスさん）

**166** 5. 例：習います → だれに　英語を　習いましたか。……ワットさんに　習いました。
- 1) 書きます →
- 2) かけます →
- 3) もらいます →
- 4) 借ります →

（絵：例＝ワットさん、1）父、2）グプタさん、3）母、4）山田さん）

**167** 6. 例：お母さんの　誕生日に　何を　あげましたか。（花）→ 花を　あげました。

- 2) どこで　日本語を　習いましたか。（アメリカの大学）→
- 3) いつ　先生に　レポートを　送りますか。（あした）→
- 4) だれに　この　CDを　借りましたか。（友達）→

**168** 7.
- 例1：切符を　買います（はい）→ もう　切符を　買いましたか。……はい、もう　買いました。
- 例2：レポートを　送ります（いいえ）→ もう　レポートを　送りましたか。……いいえ、まだです。
