# ビジネスの挨拶・自己紹介（日本語学習者向け）

---

## ① 日常会話表現とビジネス会話表現の比較・解説

### 挨拶の場面別比較

| 場面 | 日常会話 | ビジネス表現 |
|------|----------|------------|
| 初対面 | よろしくお願いします | はじめまして。○○と申します。どうぞよろしくお願いいたします。 |
| 2回目以降の挨拶 | また会いましたね / こんにちは | お世話になっております。 |
| 久しぶりに会う | 久しぶりですね | ご無沙汰しております。 |
| 仕事中のすれ違い | お疲れですね | お疲れ様です。 |
| 先に帰るとき | もう帰ります | お先に失礼します。 |
| 相手が帰るとき | ではまた | お疲れ様でした。お気をつけて。 |
| 前回のお礼 | この前はありがとうございました | 先日はありがとうございました。 |

### 自己紹介フレーズの比較

| 内容 | 日常表現 | ビジネス表現 |
|------|----------|------------|
| 名前を言う | 〜です（山田です） | 〜と申します |
| 所属を伝える | 〜の〜です | 〜の〜と申します |
| 出身を伝える | タイから来ました | タイから参りました |
| 勉強中であることを伝える | 日本語を勉強しています | 日本語を勉強しております |
| 謙遜する | まだ下手です | まだ不慣れな点もございます |
| 締めの言葉 | よろしく | どうぞよろしくお願いいたします |

### 語彙の比較（日常語 → ビジネス語）

> 同じ意味でも、日常とビジネスでは言い方が違います。

| 意味 | 日常会話 | ビジネス |
|------|----------|---------|
| ありがとう | ありがとうございます | ありがとうございます／恐れ入ります |
| ごめんなさい | ごめんなさい | 申し訳ございません |
| わかりました | わかりました | 承知いたしました／かしこまりました |
| どうぞ | どうぞ | よろしければどうぞ |
| 来ます | 来ます | 参ります |
| 行きます | 行きます | 伺います（相手の場所へ）／参ります |
| います | います | おります |
| もらいます | もらいます | 頂戴いたします／いただきます |
| 見ます | 見ます | 拝見します |
| 言います | 言います | 申します |
| 知っています | 知っています | 存じております |
| 知りません | 知りません | 存じません |
| 少し待ってください | ちょっと待ってください | 少々お待ちくださいませ |
| どうですか？ | どうですか？ | いかがでしょうか？ |
| できません | できません | いたしかねます |

**ポイント**
- 「お世話になっております」は2回目以降のビジネスの挨拶で最もよく使う万能フレーズ
- お辞儀は 15〜30度 が基本（深いほど丁寧）
- 相手より先に挨拶するのがマナー
- 日常語をそのままビジネスで使わないよう注意

**名刺交換のとき**
- 渡すとき：「よろしくお願いいたします。」
- もらうとき：「頂戴いたします。」

## ② 日常会話表現からビジネス会話表現への変換練習

### 練習1：穴埋め練習

（　）に適切なビジネス表現を入れましょう。

1. 「ごめんなさい」→「（　　　　　　　　　）ございません。」
2. 「ちょっと待って」→「（　　　　　）お待ちくださいませ。」
3. 「来ます」→「（　　　　　）ます。」
4. 「知りません」→「（　　　　　）ません。」
5. 「見ます」→「（　　　　　）します。」

> 答え：1. 申し訳　2. 少々　3. 参り　4. 存じ　5. 拝見

---

### 練習2：文変換練習

日常会話をビジネス表現に直しましょう。

1. 「山田です。よろしく。」
   → 「山田と（　　）します。どうぞよろしくお（　　　）いたします。」

2. 「また会ったね。」
   → 「（　　　　　　　　　　　　）。」

3. 「もう帰るね。またね。」
   → 「お（　　　）に（　　　）します。」 ／ 「お（　　　）様でした。お気をつけて。」

> 答え：1. 申・願い　2. お世話になっております　3. 先・失礼 ／ 疲れ

---

### 練習3：自己紹介スピーチ

テンプレートを使って自己紹介をしましょう。

```
はじめまして。（名前）と申します。
（出身地）から参りました。
現在、（仕事/勉強）をしております。
まだ不慣れな点もございますが、どうぞよろしくお願いいたします。
```

**例文**
```
はじめまして。ソムチャイと申します。
タイのバンコクから参りました。
現在、日本語を勉強しております。
まだ不慣れな点もございますが、どうぞよろしくお願いいたします。
```

## ③ 会話シーンを想定したロールプレイ

### シーン①：初対面（ビジネスイベントにて）

> 登場人物：田中さん（日本人）、ソムチャイさん（タイ人・日本語学習中）

**田中**：はじめまして。田中と申します。よろしくお願いいたします。

**ソムチャイ**：はじめまして。ソムチャイと申します。タイから参りました。日本語を勉強中ですので、よろしくお願いいたします。

**田中**：そうですか！日本語、お上手ですね。

**ソムチャイ**：ありがとうございます。まだまだです。

**田中**：いえいえ。何かあれば、気軽に声をかけてください。

**ソムチャイ**：ありがとうございます。よろしくお願いします。

---

### シーン②：2回目の出会い（社内・廊下にて）

**田中**：あ、ソムチャイさん、お世話になっております。

**ソムチャイ**：田中さん、お世話になっております。先日はありがとうございました。

**田中**：こちらこそ。最近、日本語の調子はどうですか？

**ソムチャイ**：少しずつ、上手になっていると思います（笑）

**田中**：それは良かった。また何かあれば、いつでもどうぞ。

**ソムチャイ**：はい、ありがとうございます。よろしくお願いします。

---

### シーン③：社内ですれ違うとき

**田中**：ソムチャイさん、お疲れ様です。

**ソムチャイ**：田中さん、お疲れ様です。

---

### シーン④：帰るとき・仕事が終わったとき

**田中**：お先に失礼します。

**ソムチャイ**：お疲れ様でした。お気をつけて。

---

**ソムチャイ**：田中さん、お疲れ様でした。お先に失礼します。

**田中**：お疲れ様でした。またね。

---

## ④ おさらい

### ①の重要フレーズを確認しましょう

| フレーズ | 場面 |
|----------|------|
| はじめまして。○○と申します。 | 初対面の自己紹介 |
| どうぞよろしくお願いいたします。 | 自己紹介の締め |
| お世話になっております。 | 2回目以降のビジネス挨拶 |
| ご無沙汰しております。 | 久しぶりに会ったとき |
| 先日はありがとうございました。 | 前回のお礼 |
| お疲れ様です。 | 仕事中のすれ違い |
| お先に失礼します。 | 退社・先に帰るとき |

### お疲れ様の使い分け

| フレーズ | 使うタイミング |
|----------|--------------|
| お疲れ様です | 仕事中に同僚・上司と会ったとき |
| お疲れ様でした | 仕事が終わったとき・相手が帰るとき |

> ⚠️ 「ご苦労様」は上司が部下に使う言葉なので、目上の人には使わないこと。
