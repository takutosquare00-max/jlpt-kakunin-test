## 問題1　漢字（読み）

___ の言葉の読み方として最もよいものを、１・２・３・４から一つ選びなさい。

**1**　時代の変化にともない、大学も大きな <u>変革</u> を迫られている。
1. へんかく　　2. へんこう　　3. べんかく　　4. へんがく
> **正解：1（へんかく）**　「変革」＝物事を大きく変えること。「変」＝ヘン、「革」＝カク（あらためる）。「へんこう（変更）」と意味・読みを区別する。

**2**　わが校では、新しい学部を <u>開設</u> することになった。
1. かいせつ　　2. かいぜつ　　3. がいせつ　　4. けいせつ
> **正解：1（かいせつ）**　「開設」＝施設や制度を新しくつくること。「開」＝カイ、「設」＝セツ。「学部を開設する」「口座を開設する」などと使う。

## 問題2　語彙（文脈規定）

（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**3**　新学部の開設のため、長年にわたり議論を（　　）てきた。
1. 重ね　　2. 積み　　3. 折り　　4. 束ね
> **正解：1（重ね）**　「議論を重ねる」＝話し合いを何度もくり返す。「重ねる」は同じことを積み上げる意味。「積む」は経験などに使い、「議論を積む」とは言わない。

**4**　新入生の皆さんには、悔いのない学生（　　）を送ってほしい。
1. 生活　　2. 生命　　3. 生産　　4. 生計
> **正解：1（生活）**　「学生生活」＝学生として過ごす日々。「生命」は命、「生産」は物をつくること、「生計」は暮らしを立てる手段。

## 問題3　文法（文法形式の判断）

次の文の（　　）に入れるのに最もよいものを、１・２・３・４から一つ選びなさい。

**5**　今回のタンカーの事故で流出した油が幅５メートル、長さ300メートル（　　）広がり、環境汚染が心配されている。
1. にともなって　　2. について　　3. にわたって　　4. を中心に
> **正解：3（にわたって）**　「〜にわたって」＝ある範囲・期間全体に。「幅5m、長さ300mにわたって広がった」は広い範囲に広がったという意味。「にともなって」は変化が同時に起こる場合。

**6**　絶滅する（　　）動植物の保護に関する条約が新しく作られたそうだ。
1. かわりに　　2. はずがない　　3. ことがある　　4. おそれがある
> **正解：4（おそれがある）**　「〜おそれがある」＝〜の危険・心配がある。「絶滅するおそれがある動植物」は絶滅しそうな危険がある動植物という意味。「ことがある」は危険の意味はない。

**7**　準備が整い（　　）、始めさせていただきますので、そのまま少々お待ちください。
1. ながら　　2. そうに　　3. 次第　　4. かねないので
> **正解：3（次第）**　「〜次第」＝〜したらすぐに（動詞ます形に接続）。「整い次第、始めます」は準備ができたらすぐ始めるという意味。完了後すぐの行動を表すので「お待ちください」と自然につながる。

## 問題4　文の組み立て

次の文の <span class="star">★</span> に入る最もよいものを、１・２・３・４から一つ選びなさい。

**8**　このサプリメントは ＿＿ ＿＿ <span class="star">★</span> ＿＿ ので、正しい用法を守ってお使いください。
1. 健康を　　2. おそれがあります　　3. 害する　　4. 飲みすぎると
> **正解：3（害する）**　正しい語順：「飲みすぎると→健康を→★害する→おそれがあります」。完成文：「飲みすぎると健康を害するおそれがありますので、正しい用法を守ってお使いください。」「〜おそれがある」＝〜の危険・可能性がある。

## 問題5　読解

次の文章を読んで、後の問いに答えなさい。答えとして最もよいものを、１・２・３・４から一つ選びなさい。

> 　皆さん、ご入学おめでとうございます。時代の変化にともない、大学も大きく変革を迫られております。わが校でも新しい学部の開設のため、長年にわたり議論を重ねてまいりました。そして、いよいよ今年度より新しい学部がスタートします。その第１期生として入学された皆さんは、新しく生まれた国際学部とともに大きく成長していくと確信しています。皆さん、どうぞ悔いのない学生生活を送ってください。

**9**　この話は、誰が話しているものか。
1. 新しく入学した学生
2. 大学の関係者（学長など）
3. 卒業していく学生
4. 新入生の保護者
> **正解：2**　「ご入学おめでとうございます」「わが校でも…議論を重ねてまいりました」「成長していくと確信しています」など、入学生を迎える大学側の立場で話している。

**10**　わが校について、本文の内容に合うものはどれか。
1. 今年度から新しい学部が始まる。
2. 学部の開設はまだ決まっていない。
3. 国際学部は昔からある学部だ。
4. 今年は新入生がいない。
> **正解：1**　「いよいよ今年度より新しい学部がスタートします」「新しく生まれた国際学部」とあり、今年度から新学部（国際学部）が始まるとわかる。
