# N2文法 ランダムテスト No.03（10問・10分）

---

## 問題

> 問題1は（　　）に入る最もよいものを、問題2は ★ に入る最もよいものを１〜４から一つ選びなさい。


**問1**　被災地でボランティアをした（　　）、たった１日だけですから…。
1. ばかりでなく　　2. というのに　　3. というより　　4. といっても

**問2**　この体操は体力（　　）、無理のないように行ってください。
1. に限って　　2. に応じて　　3. にかわり　　4. において

**問3**　このサプリメントは ＿＿ ＿＿ ★ ＿＿ ので、正しい用法を守ってお使いください。
1. 健康を　　2. おそれがあります　　3. 害する　　4. 飲みすぎると

**問4**　普通の社員（　　）私の意見など、経営陣は聞いてくれないと思うんです。
1. に限る　　2. にすぎない　　3. にしても　　4. といっても

**問5**　１年前、私は日本での生活に不安を（　　）、成田行きの飛行機に乗り込んだ。
1. 感じ次第　　2. 感じつつ　　3. 感じるおそれがあって　　4. 感じかねないので

**問6**　商品の受注が増えたのはいいが、それ（　　）残業が増えて、従業員から不満が出ている。
1. にともなって　　2. にわたって　　3. にとって　　4. にかけて

**問7**　こちらの車は ＿＿ ＿＿ ★ ＿＿ ので、キャンプにも使えます。
1. 寝泊まり　　2. 車内で　　3. ようになっている　　4. できる

**問8**　9月になったのに、涼しくなる（　　）、さらに暑さが厳しくなったような気がする。
1. に反して　　2. くらいなら　　3. はもとより　　4. どころか

**問9**　悩みを ＿＿ ＿＿ ★ ＿＿ しようがない。
1. アドバイスの　　2. 相談されても　　3. わからないことには　　4. 詳しい事情が

**問10**　テレビ番組は内容によっては、若者に悪い影響を（　　）。
1. 与えざるを得ない　　2. 与えかねない　　3. 与えないところだ　　4. 与えっこない

---

## 解答と解説

**問1**　正解：**4**（といっても）
「〜といっても」＝〜とは言っても（実態を修正・限定）。「ボランティアをしたといってもたった1日」は大したことではないという謙遜。「というより」は言い換え・修正を表す。

**問2**　正解：**2**（に応じて）
「に応じて」＝状況や程度に合わせて。「体力に応じて」は自分の体力に合った形でという意味。「に限って」は〜だけ、「において」は場所・場面を表す。

**問3**　正解：**3**（害する）
正しい語順：「飲みすぎると→健康を→★害する→おそれがあります」。完成文：「飲みすぎると健康を害するおそれがあります。」「〜おそれがある」＝〜の危険・可能性がある（否定的な意味）。

**問4**　正解：**2**（にすぎない）
「〜にすぎない」＝〜でしかない（程度が低い・大したことではない）。「普通の社員にすぎない私」はただの平社員でしかないという自己評価。「に限る」は〜が一番（推奨）。

**問5**　正解：**2**（感じつつ）
「〜つつ」＝〜しながら（二つの動作・状態が同時に進行）。「不安を感じつつ飛行機に乗り込んだ」は不安な気持ちを持ちながらも乗り込んだという意味。「次第」は〜したらすぐにという意味。

**問6**　正解：**1**（にともなって）
「〜にともなって」＝〜と同時に変化して。「受注増加にともなって残業が増えた」は受注が増えるにつれて残業も増えたという意味。「にわたって」は範囲・期間の広がり、「にかけて」は期間の幅を表す。

**問7**　正解：**4**（できる）
正しい語順：「車内で→寝泊まり→できる→★ようになっている」。完成文：「こちらの車は車内で寝泊まりできるようになっているので、キャンプにも使えます。」「〜ようになっている」＝そういう機能・仕組みになっている。

**問8**　正解：**4**（どころか）
「〜どころか」＝〜どころではなく、むしろ逆。「涼しくなるどころかさらに暑くなった」は逆の変化になったという意味。「に反して」は〜に逆らって（予想外の結果）。

**問9**　正解：**3**（わからないことには）
正しい語順：「相談されても→詳しい事情が→★わからないことには→アドバイスの」。完成文：「悩みを相談されても、詳しい事情がわからないことにはアドバイスのしようがない。」「〜ないことには」＝〜しなければ（条件）。

**問10**　正解：**2**（与えかねない）
「〜かねない」＝（よくないことが）起こる可能性がある・〜しそうだ。「悪い影響を与えかねない」は与えてしまうかもしれないという懸念を表す。「ざるを得ない」はやむを得ずする場合に使う。
